\n SteamCon is a "steampunk" themed con being held in Seattle on Oct 23-25 2009.

There is gaming space (two rooms) and the woman who just took over gaming is *very* open to story games & other RPGs.

The only catch is that the games have to be steampunk/victorian/weird-west/pulp/swords&plantes themed and/or set.

I'll be running a game on Sunday noon-3. Either a Victorian set InSpectres or a mad scientist romp i'm calling "To the Moon". (I'd be running more but I've been lassoed into running all the music)

If you're interested in running a game at this con please contact

Danicia:
dprior@inbox.com
