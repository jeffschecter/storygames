\n Hello all,

I'm very proud to announce that I've published a fully-playable french version of Ghost Lines.

Disclaimer: since I don't have a copy of InDesign, and that I'm a web developer, I had to build the document using "the thing I know the best", that is to say HTML. It's not pixel-perfect, but I'm quite happy with the result.

You can browse the HTML document here:

http://brunobord.github.io/ghostlines-fr/

And download its PDF export here:

http://brunobord.github.io/ghostlines-fr/ghost-lines-vf-1.0.pdf

The whole source and script used to render the document are available on the Github project page.

The good part is that, with a bit of luck, I may be able to playtest it with old friends on the 5th-6th of october, which is exciting. I'll try to make an AP report for you guys.
Thanks to everyone.
