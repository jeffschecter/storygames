    \nhttp://lapo.it/rpg/dadi.php
    Write in Descrizione the label you want to add to the roll
    Write in Dadi the dice you want to roll without modifiers. You can roll fudge dice writing XdF (where X is the number of fudge dice). If you want to roll different dice together, you had to write for example 9d6+5d10 and so on.
    Then click on Lancia to roll the dice.
