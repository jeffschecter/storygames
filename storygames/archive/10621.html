\n Any thoughts on what the "perfect I-Phone rpg book set up would be?
\nSome thoughts on my end:
\n
\n- Dice roller geared towards the game mechanics.
\n- Ability to save npc's in a portfolio on the phone.
\n- Ability to save notes on the game in the phone.
\n- Potential to easily print out the above two.
\n- Possibly linked to a social network database with a forum/chat function that allows people to play with others all around the world any time.
\n- Possibly built in recorder to send messages to one another for a more "in person" feel to long distance games.
\n
\nI would love opinions on how others think that smart phones could best be used for enhancing gaming environments on the go.
