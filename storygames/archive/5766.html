    \n Yeah, as Jason mentioned, I'm on travel with only internet cafes for another week. If you can be patient until early March, I'll get you my conversion notes, and the fixes I made after playtest. But from what I remember:

    credit = glory (you're bucking for promotion within Delta Green)


    my original notes to John Harper before he made the sheet:

    Insight => Instinct
    Grace => Agility
    Might => Strength
    Spirit => Sanity

    Heal => 1st Aid
    Lore => Dark Lore
    Music => Magic (I'm iffy about this one, if you can think of something better, do it!)
    Orate => Manipulate

    Athletics => Athletics
    Cunning => Stealth
    Hunt => Tradecraft
    Wrestle => Martial Arts

    Aim => Rifle
    Shield => Dodge
    Spear => Submachine Gun
    Sword => Shotgun


    Helmet => Helmet (-1 Rifle)
    Breastplate => Body Armor (-1 Agility)
    Greaves => Kickass Boots (-1 Stealth)

    Divine Favor => Agency Favor
    God => Home Agency
    Lineage => Concept
    Heroic Trait => Specialty
    Glory => Credit

    I didn't give names to the sets of skills - you could use AGON's or come up with your own.

    I didn't do sanity as fate, but it's certainly possible to conceive of it that way. I prefer fate just being fate; getting to the end of that track doesn't ALWAYS mean you went crazy; just sometimes. But I did attack sanity as a skill in-game - I wanted to be able to directly call for sanity checks and attack it (jason wisely bumped his up to d10 I think).

    I don't think we did skill combinations to set up certain actions, but it should be done.

    That's all I can think of here - I'll bookmark this thread to revisit when I'm back at my own computer.

    Good Luck!!
