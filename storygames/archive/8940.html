\n Folks,

I'm going to be finalizing the manuscript for A Penny For My Thoughts by the end of the month (yay!), and I want to make sure I don't forget to credit any playtesters. I think I've got a complete list, but if you playtested this game at all, please drop me a line at ptevis@gmail.com to let me know how you'd like to be listed.

Thanks,

--Paul
