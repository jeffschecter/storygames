Post ideas here, certainly, but if you want to help with the registration process, you can submit entries to the award registration page:

http://www.rpg-awards.com/2012/

I'd especially like ideas for supplements for games and free RPGs.

    \n Yeah, Joe has it here. The important issue is just that a game can't be in the running twice - new editions don't count. So if a game is in the running this year for its ashcan edition, that means that it won't be listed the next year for its later non-ashcan release.

    In practice, ashcan releases haven't usually been listed for this reason, but if a designer wants, I'll explain the choice and list the ashcan if that's what they want.

    So, catty_big - if you want to run Sci-Fi Beta Kappa for its free, pre-ashcan PDF release in 2012, you can choose to do so. But then it won't be listed for its ashcan edition this spring, or its non-ashcan edition later on.

    So before I start copy/pasting links to supplements, John, can you give me some idea of what counts as an indie supplement? i.e. I'm assuming that a Pathfinder third-party supplement wouldn't qualify...does the core game have to also be indie? But then Day After Ragnarok won that one year and it was a Savage Worlds game. (I guess this answers my question about Hael.)
    \nWell, don't treat every previous choice as necessarily correct - but the principle is the same as for indie RPGs in general. If the primary author produces and controls the work, then they're indie. I don't require strict ownership, having decided that Penny for My Thoughts is definitely indie. This is a tricky line to walk, but it's also not a tragedy if the line isn't drawn perfectly. The main thing is that I don't want to include works where freelance author(s) submits their writing, and others own the result and control how it all goes together.

    This could technically include third-party Pathfinder supplements if the author has full control over it. I wouldn't list such unless there is an outstanding reason to do so, though.
