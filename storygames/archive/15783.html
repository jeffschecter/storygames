    \n
    \n G+ features:
    \n
    \n- Ability to easily ignore specific people or threads.
    \n- Ability to get updates on a specific thread via email.
    \n
    \nGeneral:
    \n
    \n- Ability to turn off all graphic (which I can do via my web browser).
    \n- Ability to minimize the user interface.
    \n- Eliminate anything that would limit my ability to scale the text easily.
    \n- Page padding or margins of at least 10-20 pixels on all sides, even when I zoom in.
    \n- Near black text on a near white background.
    \n- Ability to hide user interface elements.
    \n- Ability to view all posts in a multiple page thread in 1 long page.

    \n No non-text ads, especially ones on the sides of the page that you can't scroll to escape.

    No user icons / avatars, or tiny ones like the ones used here.

    No user signatures. User signatures have a strange power to transform an otherwise enjoyable forum poster into a fucking See n' Say.

    "Click to load this image" functionality so images in posts are not displayed by default, but the reader can click to see them when the coast is clear.

    Text browser (w3m, elinks, etc) compatibility. A text browser running on a remote host and viewed thru a ssh tunnel is a good stealthy browser.\n
