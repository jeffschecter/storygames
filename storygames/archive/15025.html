    \n I think any of those games (I'm not sure about D&D's character growth) would work fine as long as both players simply write down the 3 most important things that happened during the session each. That way there would be at least 3-6 things for both players to grab on to for next time.

    If you have more players, In a Wicked Age might be just the thing, but some characters make get dropped from the story if they don't make a big splash on first arrival.

    If you want Sci-Fi, FreeMarket is all about memory and experience, so if the players forget stuff it could actually add to the feel.
