\n So, I may be running some Ghost/Echo for a friend via Skype, which I've never done before. I need suggestions for...

*A dice roller we can both see the results of
*Some way of tracking/displaying goals and dangers that we can both see

    \n There's actually a really simple dice roller you can get for skype. Go to Tools > Extras and type "dice" in the search bar.

    I'd suggest using MapTool for your dice roller and keeping track of stuff (EpicTable would be better but it's not quite in Alpha yet), but you're going to have to figure out the port forwarding.
