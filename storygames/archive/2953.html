\n A few months back I tinkered with the rules of Dead of Night to help make a 28 Days Later scenario to run at Conception. It ran really well, and now I'm polishing it up to release next week ... not coincedentally timed for the release of 28 Weeks Later at cinemas.

But ... I like my house and my money, and I don't particularly want to get sued! So I turn to Story Games! Help me think up a name for it so I don't get sued!
