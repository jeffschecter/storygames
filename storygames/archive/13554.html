\n A while back, towards the end of a discussion on a global gaming convention, I posited the idea of a game that gets passed around the world:\n
Posted By: Wordman
Since the convention would chase the meridian around the world, sort of like New Years celebrations, this would open the possibility of taking the results of a particular game in, say, Australia and feeding it into the start of a game in Europe, and so on. Or, perhaps, certain characters are passed through the time zones (IAWA might be interesting this way). Or something.
\nI've been thinking a bit more about the logistics of doing this. To make it work, the idea would be to run "sessions" of a certain length at spots equally distributed around the planet. Each session would contain mostly gaming, but would end with a "hand off" period.

This hand off would be time for the GM (or whole group) to pass the story along to the GM (or whole group) of the next session, and for the next session to do a bit of planning before they start. Chances are any overall storyline would get pushed pretty far off the rails, so this period would allow the next session to adapt to that to keep the story moving. Also, any logistics that might be needed could get handled here. For example, if a character gets killed in a session, a replacement character might get figured out during this time. Or maybe people could post about it or whatever before the next session starts.

The trick to the whole thing is figuring out what length of session works to place the location of each session in a spot on the globe where people might actually game.

A three hour session time would allow the sessions in eight different places, but is almost certainly too short. It also inevitably places a few of the sessions in some odd spots, like Greenland and Mongolia.

A four hour session duration would allow six sessions. Looking at a map, this sequence of locations seems like the most likely:\n
  1. Session One: New Zealand
  2. Session Two: Perth, Manila, Hong Kong or Shanghai
  3. Session Three: Afghanistan (likely would need to draft some military players for this one) or the Seychelles
  4. Session Four: London or Paris
  5. Session Five: Newfoundland or Buenos Aires (you could probably cheat a bit here and use New York, even though it is really a time zone off)
  6. Session Six: Los Angeles or Seattle or some other US West Coast city
\nA five hour session duration would allow five sessions, one of which would need to be slightly short (you could even this out by making each one four hours and 48 minutes). The benefits of this duration are that it provides a comfortable four hour playing time with a good buffer between sessions.\n
  1. Session One: Sydney
  2. Session Two: Mumbai
  3. Session Three: London
  4. Session Four: New York
  5. Session Five: Hawaii
\nA six hour session duration would allow four sessions, but this is probably too long. Also, I don't see a good way to line up the locations, without at least one of them being in a weird spot or missing a fairly large concentration of gamers. About the best I can come up with is:\n
  1. Session One: New Zealand
  2. Session Two: Somewhere in India or Burma
  3. Session Three: London
  4. Session Four: Chicago
\nAnother possibility would be:\n
  1. Session One: Sydney
  2. Session Two: Afghanistan
  3. Session Three: Sao Paulo
  4. Session Four: San Francisco or Portland or some other US West Coast city
\nAny preferences?

Also, what game(s) would lend themselves to this? (Consider that this could be a multi-day event, with the story rounding the world several times.)
