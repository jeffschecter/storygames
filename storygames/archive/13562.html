\n So over on Mr. Elliott's site Zozer he has a document in the Traveller section called Travelling Light. In it is a slightly modified skill system which is cool (better then the static 8+). Also a section called Broad Skills, Basic Chargen which sketches out how to go about swapping in skills tables from Books 4-7 into the basic tables of Books 1-3. There are a lot of tweaks that can be done to give different feels to the Traveller Lifepaths.

I put together one version
Skill Tables google doc version
