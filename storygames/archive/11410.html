\n On Friday night, I\u2019m organizing a secret, invite-only session of this crazy social game. I did not design this game. I can\u2019t tell you the name of the game or the rules, only that it\u2019s a trading/bargaining game with an interesting twist. I need 20-30 participants to pull this off. I want to run it Friday from 12-2 am. It's an absolute must for the game designer. Let me know if you're interested. If we get critical mass, I'll run it.

Also, my two other event submissions \u2014 a Mouse Guard game and a Burning Wheel game \u2014 didn\u2019t make it onto the schedule because of a glitch in the system. I\u2019ll try to organize these games via twitter so we can play regardless.

You can follow my twitter feed (Burning_Luke) for updates and locations.
