\n We have Nearby and Meetup.com for face-to-face gaming and that's cool. I know there is a big community for IRC games, too. Wouldn't it be great to have some forum to post for Skype or other VoIP games? I recently joined a Yahoo Group (thanks DaveC!), but I still think a meetup.com type of setup would be great for Skype games. So here are my initial thoughts...

I figure the vital pieces for a participant entry would be as follows:\n
  • Name or Handle
  • Timezone
  • Availability
  • Game Types of Interest
  • GM or player preference
  • Languages spoken (and fluency)
  • Online Game Tool Access
\nIf you were to list a game on this imaginary site, you would need to detail the following:\n
  • Name of Campaign
  • Number of Openings (or maybe you could have an application process showing slots remaining)
  • Language Spoken
  • Game System
  • Blurb on Game setting, story focus
  • Timezone of organizer
  • Game time (or times if you are scoping for players)
  • Frequency of game
  • Online tools used (online group, wiki, forum, email, etc.)
  • Method of contact to join
\nI think this would be a great way for SG Designers to run games in development, run games to give PR to their designs, and to spread the word to those gamers who have no face-to-face group (like yours truly). I know caxino has plans to be an SG online game playing tool, but this could work in coordination with caxino, to actually feed players into the system.

What are your thoughts, SGers?
