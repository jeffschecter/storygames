\n The Story Games Names Project is a great resource, and it's been great to have it at my table. (I even converted it to a sort of ePub/Kindle format.) I was thinking that I would find it personally useful if some of the content here was relicensed to a more permissive format so that this material could more easily be remixed into commercial works - so, from its current CC-BY-NC-SA to a CC-BY license.

My request: if you have contributed to the Story Games Names Project in the past, please be in touch and let me know if you consent to this relicensing. If you're not sure, you can check with me about what material I have contributed by you, and of course there is no demand that any one relicense. I'm dev@forgreatjustice.net. (I'll personally contact anyone else who hasn't responded after a few weeks.)

The ostensible upside of this would be establishing this body of material for dropping these name tables into whatever commercial story game, app or project you're putting together. I also have ideas of using this as a base to update an expand on the SGNP once I've relicensed what I can.

Thanks!

    \n Yes, on both my contributions (the Egyptian names list and Japanese names lists; however I might want to rework my lists a little, as I've found some badass fun names since that time; please tell us a deadline). The only issue is that (and I don't have it in front of me) my Egyptian part was combined with another author's IIRC into one giant list. If they're not responsive to redoing the list attribution, I think that somewhere I have my original Egyptian data.

    Thanks Dev: This is something that Jason and I have been kinda waffling about starting up for a few years now, I really appreciate you stepping up to make it happen.

    If you can't get contributors to respond (or they respond in the negative), hit us up here; I'm sure some newer members would love to have a hand in rebuilding this artefact.

    As for how I use the SGNP: I have made it a table reference for almost every single game event -- campaign or one-shot -- since its release. When creating NPCs on the fly, someone at our table will utter the Polaris-like ritual phrase, "If only someone had thought to organize fictional character names into easy-to-reference lists for role-playing game", which to say requires about the same length of time required to stand up, go over to the bookcase, pull it out, return to the table and reference a name.
