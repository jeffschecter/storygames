\n I'll be bringing a metal easel and 2 large green foamboards
This will become the G.o.D. Meet-Up Board for gaming.
If you want to run a game for later in the day, Post-Its and Markers will be provided.

Write down the name of the Game you want to Run and for how many players and when you want to run it.
then Stick it on the Meet-Up Board.

I've picked up Arrow Shaped Post-Its for after Hours Meet-ups.

I'm hoping this will be a great way of bringing games and gamers together.
