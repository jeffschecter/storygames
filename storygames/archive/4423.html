\n This is a half-baked idea I've had in the half-bakery for a long time. There exist several Nomics out there whose purpose is to create a board game; the one that's probably the best known is Nomopoly (using a Nomic to create a Monopoly-like game). I've often wondered about using a Nomic similarly to create a roleplaying game.

    \n Actually, yeah, my roommate and I in college tried to do this as an exercise, twice. Basically: "The game is a blank slate. As soon as you want to do something for which there should be rules, let's make up the rules on the fly!"

    The problem is:
    * Not one person's vision. Lots of fights and back-and-forthing.
    * Everyone presents rules for games that they're already seeped in. Nothing new is made. When the dust settles, you're looking at World of Darkness with Two House Rules, or the same for Another Game System.
    * It's more painful than it sounds. Seriously.

    But if someone sometime pulls it off, I'd like to see what they come up with.

    -Andy

    \n A few points:
    \n
    \nA) We discovered Nomic during the process of creating Universalis. I think either Ralph stumbled across it, or one of our playtesters brought it to our attention. Also Ian Millington's work on "GM-less" play. Neither was highly influential (certainly not our inspiration), but both had some small effects in the end, I think.
    \n
    \nB) There are two categories of Nomic. I can't recall the names for sure, but I think that they're "Pure" and "Constitutional." In Pure, you start with nothing at all, no basis. With Constitutional, you start with some basis, some starting rules that are agreed to by all of the participants. This latter form, I believe, then has to sub-forms... one where you can change the starting rules, and another where you cannot.
    \n
    \nC) Also see a game called Evolve. Or maybe it was Evolution? But a similar concept. Actually, I've lost it... if you can find it out on the intarweb, I'd love to find it again.
    \n
    \nD) While, yes, Nomic is intended to creat a game that can be won or lost, I think that "RPG Nomic" would simply be an attempt to create a set of RPG rules on the fly. Dropping the regular Nomic ideal to win. Call this a constitutional form that does not vary after start, if you like.
    \n
    \nMike
