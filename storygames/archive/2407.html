  • jenskot
    February 2007 edited February 2007
    Jason, I thought this at first as well. But if you don't, it will appear to track several of your second level pages as it follows basic links but it won't do an accurate job and it won't track any third level pages at all or second level pages that visitors directly visit without first entering via your first level pages. Site Overlays, length of visit, exits, user activity and much more won't be properly tracked.

    http://www.google.com/support/analytics/bin/answer.py?answer=40932#0.1.1_step5
    Step 5 - Add the tracking code to your pages

    Google Analytics only tracks pages that contain the Google Analytics tracking code. You'll need to add this code to each page of your site, either manually or through the use of includes or other methods.
    http://www.google.com/support/analytics/bin/answer.py?answer=26908
    How do I add tracking code to my website?

    Copy and paste the code segment into the bottom of your content, immediately before the </body> tag of each page you are planning to track. If you use a common include or template, you can enter it there.
    It also depends on how your website is setup. If you use includes, if it is database driven with pages all spawning from a single template and so on. For example, on a PHPBB site, I only have to add the code to a single page and it will track the entire site.
