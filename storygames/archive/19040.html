\n My new live action game The Climb is now available. This is a short, intense game for six participants. It's been heavily playtested and includes a fantastic atmospheric soundtrack produced by Kit La Touche, with original art by Brennen Reece. Replay value is high and setup is easy. If you like my tabletop stuff but have never tried live action roleplaying, this is an excellent place to start - there will be sacrifice, heartbreak, death and betrayal!\ufeff

Everything I do is informed by tabletop sensibilities, and this project was no different - I think it is very accessible and I hope you'll give it a try.
