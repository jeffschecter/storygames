\n Hi there,

After a long hiatus away from the Story Games community, I've finally could catch up and go back to you all.
While I was digging the web to look for the tiny-est / smallest / most minimalistic RPG possible, I've started to think:\n
Would it be possible to design a (playable) RPG system that would fit in a tweet?
\ni.e.: 140 characters or less, no more.

Considering the challenge, and after a few iterations and different configurations, I've ended up with:


STR/DEX/INT spread -2,0,+1
+1:Warrior/STR,Thief/DEX,Mage/INT
HP:6+STR+lvl
Act:2d6+stat(-1/HP<6). 6-:KO, 7-9:so-so, 10+:OK
Spell: INT+lvl/day


Lots of details are missing, I know, but the core rule is here: you can create a character, assign stats, roll dice to solve actions (the AW pattern is very helpful here).
For wizards, you know how many spells one could cast per day... but nobody knows which spells. That'd be up to the GM to build spell lists.
IMO, the most obvious missing item would be the "combat rules". But if you want to stay minimalist, you could state that a success means you win the fight, a failure that you're hit, and a partial success that both opponents are hit.

There's a twitter account: @140cRPG.

There's even a "class-less" version:


Lvl: 0
STR/DEX/INT (-2<=stat<=+1, sum==0)
HP: 6+STR+lvl
Act: 2d6+stat(-1/HP<6) 6-:KO, 7-9:so-so, 10+:OK
Spell: INT+lvl/day if Magic-aware


Lots of things ahead:

* Variants: Star wars, Chtulhu, Zombie, Cowboys, etc...
* "Advanced rules" that would develop the "fuzzy" areas (Fights, Magic, Levels). My intent is to squeeze each section to a maximum of 140c,
* Build a website that would explain the game, give explanations on how the core and extended rules work. This part is WIP.
* And of course... Play this game. ;o)


Maybe the story doesn't stop there. I don't know if this community has already been challenged with a constraint like this one, but maybe some of you may be tempted to design a "140c Role Playing Game" core ruleset? Would some of you publish it on twitter? Try to play it? Hack another one?

(There might be some questions like: "Why would we do that?" -- My answer would be: "well... because we can?")

    \n I did something similar for my Game Chef entry.
    I split it up over a few tweets. See them as the D&D core releases.

    "Starter Set"
    Choose a DM, the rest play characters. You need d20 and d6. Let your imagination run wild. Absorb what others say. Improvise if needed. #OSR

    "Dungeon Master's Guide"
    DM: You describe the world and play NPC & monsters. For checks set a target number. Players need to roll high. Give out LeVeLs betimes. #OSR

    "Player's Handbook"
    PC start at LVL 1. Choose a class. For checks roll d20+LVL. If it fits your class add 5. Attacks do d6+LVL damage. PC get 5*LVL health. #OSR

    "Monster Manual"
    Check against enemy LVL to hit. Attacks do d6+LVL damage. Class may add +5 here. PC attack first. Damage reduces NPC LVL and PC health. #osr

    (The titles aren't part of the 140, instead of the #OSR abbreviations could be used possibly, like PHB)

    \n wow, these are excellent creations, I'm really glad that I'm not the only one to have been tickled by this challenge.

    May I ask you all something?

    1. If some of you are on twitter, would you like to follow the little (tortuous, but the only "safe" way to get these creations in full) procedure? (no obligation, though, do it if you want...)

    a - create a tweet with your creation (@James_Mullen and @Biest if you could find the original tweets, that'd be great too!)
    b - go to this tweet page (click on Detail) and copy-paste its URL from the address bar
    c - send a new tweet including this address targeting the @140cRPG account

    I'd be very happy to retweet any creation / hack and spread the word!

    2. As I'm in the process of building a website to explain the original creation and make a catalog of variants / hacks / spin-offs, I may include some of the great RPGs in this thread.
    Of course, I would only include games if his/her author tells me to ; the author will be quoted and every creation will be published as "all rights reserved" unless specified otherwise. Please tell me if you're interested.

    Thank you again for your ideas and suggestions, and keep it running!

    (BTW, I've cross-posted this 140c RPG concept on a french-speaking RPG community forum, and I've had excellent suggestions, some of them have even been published as a PDF in a business-card-sized-frame ; one could carry the RPG rule around in the pocket. Wicked!)
