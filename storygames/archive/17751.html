Hey Story Gamers,

I just got done prepping the site for the pay-what-you-want End of the End of the World RPG Bundle that I'm doing to celebrate the incoming new year. It features three of my games - God-king, The Legerdemain Betrayers, and Haunt Me. There's also two playtest drafts and high-resolution wallpapers included with the bundle for anyone interested in game development (seeing where God-king and The Legerdemain Betrayers started out) or desktop decorum, respectively.

For me, the coolest thing about the bundle is releasing my newest game, Haunt Me, which I've been brewing for a bit now. It's a very short, experimental game that focuses on a group of players that is forced to discuss the guilt they harbor to a bunch of strangers whilst in a haunted house. It would be really great to hear you all think about it.

You can get a look at the bundle over on my site: http://spiffup.org/endoftheworld It'll run until January 2nd.

I pulled an all-nighter for this release (hit a lot of last-minute bumps) so I apologize in advance if there are any hitches with the site or the bundle itself. Please let me know either here or via email (danielcruzchan@gmail.com) if there's any problems you run into.

Thanks everyone and happy new year!
-Daniel

    \n Hey all,

    Just as a heads-up, the problem with extracting the contents of the .zip file for the bundle has been sorted out. I was using a compression method only supported by newer software and, even at that, still specific ones only - this explains why some people could extract it while others could not. I went ahead and used a legacy compression method so the bundle should now be fully extractable by any program you use to do that. I would like to apologize to those of you that have had problems with the zip file. You can use the same link to re-download the file in order to get the updated version - if you've lost that link just let me know.

    Anyway, I hope everyone has been having a happy new year. As a reminder, there's less than 12 hours left to get the bundle if you still would like to check it out!

    Thanks for all of your support, Story Gamers!
