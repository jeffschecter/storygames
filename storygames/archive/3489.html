\n So Agon is the current bestest game ever. At least, that's what we think.

My players are really digging the "Battles which are not really battles, but for which the metaphor of battle works very well".

The only problem we're having is this:

The rules say quite simply that characters taking "wounds" in a non-combat, non-physical still takes wounds. Which is fine, I can totally dig why a severe knock to a hero's ego in an argument fucks their shit right up. The problem comes when they try to remove that wound and still use healing to do so, which kinda breaks the fiction of the thing.

So we came up with two solutions to this, and I was wondering if anyone else has tried or thought about things like this. The first is just to have players heal "spiritual wounds" with different attributes. So using, say, Orate or Music instead of Healing. The only problem there might be that this makes those attributes a bit too powerful perhaps?

The other option is to have those injuries convert into impairments, which are then dealt with in the usual way.

Anyone tried either of those or got any thoughts?

    \n It would seem like an easy rules hack if you wanted to experiment with "realms of harm", each with its own healing skill. So you might say that the realms of harm are physical, spirit (composure) and social. When a wound is received, mark the would box with a letter to indicate the type of harm. Each realm will have its own healing skill, for example Heal for physical, Music for composure, and Orate for social. Then when healing, the healing contest removes the highest wound of the type appropriate to the skill being used, plus one OF THE SAME TYPE for each additional victory.

    So for example if character's wound track was filled as follows: S C C P

    And another character sang him a soothing song, and gained an extra victory, the wound track would now be S [ ] [ ] P

    (where [ ] means empty box).

    Hey, let me us know if you try something like this, and if it works.
