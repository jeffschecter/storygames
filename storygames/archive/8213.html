\n Hello,

I've been working on making a detail-oriented RPG database, you can see the work in progress Here

I'd appreciate some input in terms of features, new list criteria, games you'd suggest for entry or added detail to existing games.

Here's a few questions I'm pondering right now:

1. How should I express something being in the "Military-" genre, as in 'Military-Fantasy" or "Military-Sci-Fi"? A obvious determining factor would seem to be whether the focal characters are military figures, but it doesn't always hold true (I wouldn't characterize Polaris as Military-Fantasy, though the characters have some standing in a military.)

2. Does anyone care about "Character Power Level" anymore? Is it a simple, one-dialed question?

3. What sort of criteria would "Post-Apocalyptic" fit into? How would you define non-Post-Apocalyptic games?

    \n Maybe instead of carefully defined genres, you should try going for "tags": Instead of labelling something "Military", maybe go for a tag system so that a single game can be labelled several things at once.

    Dungeons and Dragons 4e: Fantasy; Strategy; Heroic

    (etc)

    You can use tags to describe the kind of system that you use as well:

    Single Die; d6-based; d20-based; Multi-dice; Cards; Tarot; Diceless; (etc)
