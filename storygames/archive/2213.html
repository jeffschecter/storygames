January 2007 edited January 2007 in Meetups / Conventions
Well I've finally got cracking and working on the project in a more driven manner. Here is what it is looking like (targetting) so far:
- A totally session, server-side driven community software.
- Supports virtual everlasting tables online, which can hold cards, sheets, notes, tokens, boards, etc.
- Written in C, compiled to an executable .cgi on the server itself (for speed).
- Everything is rendered as rather standard html, with javascript support for interaction.
- Features badges (online identity), Tables (game spaces), and a Lounge (slow but more feature-rich chatting).

At its current state, it can only register and login users. I'm posting this here because for the alpha test of the software, I want 100 people. In other words, the first 100 people to register on the Caxino site will get in and the rest won't be able to register until beta testing. Of course I many up the number as things are proven stable and functional. All that I ask if you register, is that you give me actual input on the site itself as it progesses. It won't be for 2-3 weeks until it actually works in any real capacity, though the registrations will stick regardless.
