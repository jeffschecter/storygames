\n So I have a Vietnam game I want to run in a couple of weeks, but I neglected to buy carry from the OrcCon IPR booth, there doesn't seem to be a PDF available and I'm in New Zealand, so having a physical copy shipped would be both painfully expensive and potentially too slow.

So, denizens of Story Games, does anyone have any recommendations for a more accessible rule set which could be tweaked to emulate a brutal character driven Vietnam movie (Apocalypse now rather than Rambo)?

Trad or Indie sugestions are welcome, but I do want a system which can be quickly explained and groked. I recently saw a thread (somewhere) using The Mountain Witch for this sort of thing, so that's an option I'm aware of.
