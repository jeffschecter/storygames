\n Hi, there, S-Gers!

So, Thou Art But A Warrior has sold all of the copies in the first print run. They sold a bit quicker than I was planning, so I'm not quite finished making changes to fix the printing issues that occurred with the first print run. But I deemed it more important to get the second print run done right than done quickly. I do apologize if that inconvenciences anyone.

That being said, the second print run is going to press imminently. Once IPR has copies, I will also start to sell PDFs through my website.

Thanks to everybody who has expressed interest!
