    \n Ok, so, this is a bit off topic but from what I've seen of Aces and Eights? It's WAY more complicated than The Burning Wheel. The Burning Wheel has subsystems that are only used in very specific circumstances, and you can ignore them until your comfortable with the basic game mechanics. If you can handle Torchbearer, you can handle Burning Wheel.

    \n I've run it. It's very old-school in terms of mood and style, but it has some very cool ideas. The basics are pretty straightforward and you can run a game using a small subset of the rules. It has a nice idea in the career paths - inasmuch as you can choose a career at any session and gain points for following that career. The sub games are great fun, and weirdly my only criticism of the game is that there aren't enough of them (I'd love a running-a-hotel subgame).
