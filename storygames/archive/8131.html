\n Next Semester, I'm going to pitch a game to the campus gaming club. I've decided its unlikely that a non-WoD or D&D game will get traction, so I'm shooting for a game of D&D combined with a bunch of ideas from Lost, the series. I'd like to get some commentary as I go through the process setting up ideas for the game:

For starters, in this game, the world as a whole is completely normal medievial world, humans only, no-overt magic, but the characters are ship-wrecked in an Island that permeates D&Dness (Many of the characters are turned into demi-humans, bullshit monsters roam the land, magic suddenly 'works', the strewn about cargo is gifted with magical powers.)

I'm totally using lego minis for this.

I'm a little bit curious if there are obvious D&D methods to escape the island - Wizard spells, for example. I'll have to think of some way to make that interesting and challenging when it comes up.

I'm using a hack from the Mountain Witch I wanted to try out: After meeting characters, NPC or PC alike, you can put points of Trust in them. These points of Trust can be used with special "Alliance" powers, which are taken off of Defender abilities based around direct help and Leader abilities based on cooperation. They can also be sacrificed for "Manipulator" powers, which are largely taken from the Rogue abilities, but suped up according to how many Trust points are burnt.

Which also puts in reasons to throw Dark Secrets in, which are very appropiate.

I'd like to throw in Flashbacks in a big way, something like, player sets up a situation in flashback and GM takes it from there, make it interface with the rules for Dark Secrets somehow - Flashbacks have to precede Director Authority? If someone has a more concrete idea, I'm all ears.

I'm also wondering whether to implement an "Others" like situation - At the simplest level, the Others are kind of fun, but at the same time, it might be hewing too close to the events of the series, as well as the fact that they can represent some of the cringe-worthy aspects of trad gaming foes (I planned you to react that paticular way to my actions!)

    \nFunny you should ask for that. I've been using 4e Skill Challenges to do exactly that for some time now. Each session, I ask the player a question about one bit of a PC's backstory, via extemporaneous scenes, each of which leads up to a roll in the Challenge. At the end, we learn the answer to the question, and the PC either gets XP for the challenge and some small bonus or they get a quest to achieve later in the game. It works really well, so far.

    \nThat's as simple as to use some arcane type of ward against magical teleportation and a massive plannar anchor effect all over the island.

    And you can even say it is in the rules! In fact, there is even a precedent. In Forgotten Realms, teleport spells don't work/are too dangerous to use when within the Underdark.

    So, going for the "Lost" style, I'd say there is a misterious magical radiation at the island, that makes all attempts at teleporting unavailable. And common (very high level) monsters can block people's attempts at using more mundane methods.
