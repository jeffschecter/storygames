    Not following how to do this? Or not following a new style sheet to make blockquote pretty? The former, switch to html and type the following:

    &lt;code&gt;
    Stuff you are quoting.
    &lt;/code&gt;

    Or you could type blockquote instead of code.

    <blockquote>
    Stuff you are quoting.
    </blockquote>

    Edit to fix it so the code code would show up in the code area, and I added blockquote.
