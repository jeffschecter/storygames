    \n To fix this:

    1. Open "bigcount.css"

    2. Locate:
    ul.DataList div.Meta strong {
    \tbox-shadow: 0px 0px 2px #777;
    \t-moz-box-shadow: 0px 0px 2px #777;
    \t-webkit-box-shadow: 0px 0px 2px #777;
    \tposition: absolute;
    \ttop: 24px;
    \tleft: 668px;
    \twhite-space: nowrap;
    }

    3. Change "left: 668px;" above to "right: 260px;"

    I don't know where else this CSS is used so it might have unintended effects.
