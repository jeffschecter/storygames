\n...and NOW what?

This is a forum freeformish game, inspired by Gregor's thread on solo gaming. Anyone can join in. It goes like this:

- Someone writes an action in the first person ("Action: I activate the starboard sensors and...").
- Someone else writes what happens - how the world/environment/fate/narrative reacts. ("Reaction: As you hear the first PING, you notice that...").
- Someone adds a detail. ("Detail: That PING reminds you of your old alarm clock. How strange.").

If two people write the same sort of post (two actions in a row, for example), the next person can decide which one(s) to incorporate and follow up on. So if I post an action at the same time as Billy Bob, Sally Sue can decide - when posting a reaction - whether both actions took place, or only one of them.

Okay? And as I said, anyone can jump in at any time. It's always someone's turn, and you can be that someone whenever you like.

I'll post a first Action to kick things off.

Edit: Make sure to start your posts with the word "Action", "Reaction" or "Detail". That makes it easier for the next person to know what to post.
