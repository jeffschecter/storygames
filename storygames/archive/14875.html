\n Just wondering if anyone else goes to the trouble to automate their games with spread sheets and other gizmos.

I recently created some spread sheets to help me with a solo run using Mythic Game Master Emulator and Magical Burst (3rd draft.) I'm rather pleased with them since they allow me to spend more time writing and less time fiddling with dice or flipping through tables. You still need the Mythic GME book/PDF to properly utilize the sheets. To actually use the features, you'll need to enable macros.

GME_test.xls (for older versions of Excel and compatibles)
http://www.mediafire.com/?3bv6rj3booemj4e

GME_test.xlsm (for MS Excel 2007+)
http://www.mediafire.com/?a4a1too6wpvztae

An example of the the first scene that I made (no need to enable macros to just read it.)
