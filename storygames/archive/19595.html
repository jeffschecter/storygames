\n Hi Everyone! To those of you that want to playtest Synthicide but don't have your own gaming group, let's try to set up several google hangouts. Post here if you're interested in joining.
The original thread introducing the game is here:
http://www.story-games.com/forums/discussion/19584/new-to-story-games-looking-for-play-testers

I've come up with some "best practices" for online play testing Synthicide. Here's what I came up with:

1. People should use google hangout, with the GM using the desktop sharing feature
2. I created a simple play grid that supports adobe reader: https://www.dropbox.com/s/35jjem0e08d2p0e/GameGrid.pdf
3. The GM uses textbox markup tools to create game objects, and the drawing markup tool to draw elements on the game grid

If you know of a better way, I'm open to suggestions.
