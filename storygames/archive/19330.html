\n"Some Godforsaken Cornfield in Pennsylvania: Spring 1778"

This is a game I ran last weekend at the Seven Years War Association Convention. It is a battle game but uses a Matrix game to run movement and a card combat game (that focuses on words rather than numbers) to run the fights. It is broken into three parts, running around 40 minutes.

Can you let me know if you've seen any other games run like this? I want to get an idea if it is new or not.

Thanks! Chris Engle

    \n The video makes it look slow but it kept moving without the long delays that happen in war games when they switch from moving to combat. When I ran it a second time w/o video, so I could play, it felt fast.

    The game was fun for all of us. As you can see it was also fun for people walking by. The transition from matrix events to combat was pretty smooth. If I'd used the scene cards from FTAL it would have been a true story game.

    I think that this approach might work for your kind of games.
