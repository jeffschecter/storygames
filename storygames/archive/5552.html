\n Can people recommend games ...

1) That play in 2-3 hours
2) That don't require the same players next session
3) That are way fun
4) That don't require people to sit in the same place (i.e. I don't have a jenga tower online).
5) That can be explained quickly and play immediately
6) That have zero prep beyond someone at the table (GM typically) knowing the rules

    \n Shameless self-promotion:

    Dread: The First Book of Pandemonium. Some folks say it's "Whispering Vault with testicles", others call it "Delta Green directed by Clive Barker."

    1) I run it at cons all the time. Usually makes for a good 2-3 hour pick-up game.

    2) Player characters are expendable. A player character who's been around for a couple of months is a veteran. They buy their milk by the pint. So, no, they don't have to be around for the next session.

    3) I think it's pretty fun. Here's the reviews.

    4) It's a d12-based system. While we don't all have piles of d12s lying around, there's also some nifty online dice rollers.

    \n I'm going to recommend my game Dirty Secrets with a couple of caveats.

    First, the basic system uses Liars' Dice, which might not be that fun online. I mean, if you trust your fellow players to roll dice and report results, it would work fine. I just wonder if it will lose something if you can't see the other person.

    Second, the game uses a Crime Grid, where you have a pawn that you move around, marking names in squares. This is a fairly significant part of the game. If you're just using Skype, this won't work out for you. However, I could see hacking this using a Google Docs spreadsheet or something.

    Seth Ben-Ezra
    Great Wolf
