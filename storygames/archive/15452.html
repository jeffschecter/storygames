    \n A few people have asked me about how we run this so I started working on this document: http://bit.ly/v2tS7t
    \n
    \nThough I've only just started it...
    \n
    \nIt is all played online as a constant ongoing thing.
    \n
    \nWe use this cool die roller: http://www.xn--metstbchen-eeb.de/cgi-bin/DIESERVER/roll.pl?diestring=&comment=&code=exFATE
    \n
    \nThe aspects in play are all at the bottom of the page for each episode, generally they are created on the fly based on things people have written. Invoking/tagging/compelling is all done in the Comments on the right-hand side of the document.
    \n
    \nThe best part of running this game is checking in and finding someone has written something awesome, perhaps taking things in an unexpected direction. It's kind of addictive, I use a add-on for Google Chrome so I know if there have been any changes to the document, and find myself checking up on things several times a day.
    \n
    \nThe most challenging part of running a game like this is actually getting it off the ground and finding some interested players. Previously I've found things just get bogged down in discussions before you have even started, so for this I tried the "Just dive right in and see what happens" approach and it seems to be working out so far.
