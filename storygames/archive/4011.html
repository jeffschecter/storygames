    \n We used iStockPhoto for Dirty Secrets, so I checked the license. Sections 3 and 4 are the relevant ones. The POD thing is for stuff like T-shirts or mousepads. Essentially, they don't want you to use a single image and slap it on a coffee mug, because then the image is really what is being sold. They also don't want you to use the image in a template or somesuch thing that would be resold. Books are expressly included as a useable option in Section 3, so you should be okay. Oh yeah, if your print run goes over 50,000, then they want some extra money. So, if you think that's a possible issue.... ;-)

    We found iStockPhoto to be a great resource for Dirty Secrets, and I'd encourage other folks to consider using them, if photography would fit the feel of your game.

    Seth Ben-Ezra
    Great Wolf
