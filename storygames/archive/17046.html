    \n So I'm well aware that Design By Committee is a recipe for mediocrity, so I'm going to try to shepherd this thing a bit, through a personal vision which will become evident... uh, as I fully form it. Which is part of the point of this post: meta-crowd-sourcing (i.e., crowd-sourcing the vision).

    Top Question So Far (polling):
    * What time period?

    I see either a modern state fair, slightly urbanized (e.g., the NC State Fair takes place at a fairground and arena on the border of the state Capitol, and is a locus of gangs and grads, families and freaks) or something more midwestern/deep south, set in the mid to late 1930s (risk of stereotyping is high; but this is, I think, what one "sees" when one hears "State Fair"). Modern is easier; historical will require a bit of research, if only for flavor notes.

    ----
    Now, on to my already known design goals...

    Elements:
    I'd like to see some really juicy and/or subtle ones, here. No punting with "Father and son" relationships or hamhanded "100 pounds of TNT" objects, OK?

    Formatting:
    I've got notes in the doc already, regarding basic type and punctuation (i.e., copy BPG PDFs). I'm just noting here, too, that commentary within the doc should be done in a color other than black, so we can see it easily.

    Intro:
    I think anyone and everyone should take a stab at the Intro, then we fine-tune the "best" one (assessment of "best" TBD--probably Ranked Pairs vote). But the Intro can't be even broached (nor many Elements, really) without answering the question of time period.

    Thoughts?
