    \n Maptool is just a dumb tool (or a very smart one), not affiliated or attuned with any system. It basically allows a bunch of people to network to the same view of maps, boards, etc. It has chat and die roller, etc.
    \n
    \nSome things that you can do with it that would support Savage Worlds is use the gridded map. Show tokens (which you can define stats on), roll dice, roleplay in chat or via Skype, etc. You could easily make a chase board.
    \n
    \nSome things you can do with a bit of macro programming is set up the initiative cards, exploding dice mechanic, etc.
    \n
    \nWhat I think is cool about this tool is thinking about non-standard grid combat ways to use it. For instance, I've come up with a bunch of generic figures and background scenes (think a static scene of a city like the old point and click PC adventure games like Kings Quest). No combat, but you can show characters in a scene and relative position. Used this for a short test with In A Wicked Age and it worked well.
    \n
    \nSince Maptool is free, you might want to download and take a look.

It makes running the game pretty easy. If you're going to Gencon this year, I think Digital Adventures (who produces all the add-on rulesets for FG2) is selling FG2 with the SW ruleset bundled at a slight discount (I seem to recall reading that somewhere recently). If you need it soon it's worth the $$$ IMO (I've used it both as a GM and currently as a player).

    \n There is a MapTool FrameWorks available for Savage Worlds (I believe it was actually approved by Pinnacle as well). It makes running a SW session a breeze. Both MapTool and the SW FrameWork are free to the public, and the guy who developed the FrameWork is a great guy who worked pretty hard on it. I have played in a game using it and it made the game very smooth and Fast!Furious!Fun!

    You can download it here:
