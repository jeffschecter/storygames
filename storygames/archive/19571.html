\n I've done a generator for the RPG Mythic few days ago, it may interest people in this forum.
You can fin it here: http://www.gamersassociate.com/mythicpick/
There is quite no layout for now.
It handles the 3 dice rolls in Mythic: random event / open question, scene interruption and yes/no question.

I'm using it for inspiration while making game or scenario or else.
Ask a question in your head (or at loud if you like to hear yourself), click on 'Random Event' and interpret the result Action + Subject with your question.
Example:
Why am i posting these here?
Random event: Abandon Tactics
Interpretation: Because I don't care any more of any release plan for my stuff.
Enjoy if you can.

The Mythic RPG can be found here: http://www.mythic.wordpr.com/page14/page14.html
I only use the Game Master Emulator which is great.
