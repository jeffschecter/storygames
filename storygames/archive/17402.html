\n So this week I'm testing out my Blood and Honor rules for being tough cops in a crazy, corrupt city.

The main thing I'm doing with it that is not in Blood and Honor is taking out the Season economy but inserting an Evidence economy. Cops need to generate Evidence to get convictions, and they need to generate enough of it before the public loses faith in them (the Integrity pool drains).

These are two documents that I've done kind of in playtest/pregenerated character format so that I can focus in on what I want to test this week. So there are some contradictory things in here and it won't make much sense without Blood and Honor.

Oh, anyway, buy Blood and Honor, it's only $5, and it's fucking amazing.

Anyway, here's my packet outlining how the game is played. It's in a weird order because it's the order I'm going to teach/go in at the first session, not necessarily the order of how the game actually goes.

Then here's the playset/pregens with first-draft character sheets and some character secrets and such.

Hope you all enjoy it.
