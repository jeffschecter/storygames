    Some more suggestions (for pay):
    - ConceptDraw is available on both Mac and Windows, and is a fairly decent vector package; however, it ain't cheap (although it's cheaper than Illustrator). It's 299 for a single license (although it might have educational pricing).

    - CorelDraw used to be significantly cheaper than it is today: now a single license costs almost 500 dollars.

    - Macromedia (when they were themselves) once made a decent illustration tool called Freehand; then they got purchased by Adobe. You can still buy the last version of Freehand (Freehand MX) from Adobe for 399. You can also get an upgrade to it for 99, so you might get more mileage buying a used copy of an older version off Ebay, and then upgrading.

    Some more suggestions (for not pay):
    - Karbon14 is the KOffice vector graphics package, for Linux, if you want an alternative to Inkscape. I was not really happy with either package when I first tried them, but that was a few years ago.

    - Skencil is a vector drawing package, written in Python (so, it's probably pretty portable). I have never tried it.

    So... for the past 15 minutes, I've been playing around with Lineform.

    \n

    It's pretty damn cool, to be honest.

    \n

    It uses all the core components of Aqua, and I'm fairly sure it's Cocoa based, so everything is very "Mac." For example, all the fonts are handled through the good ol' system wide font panel (you know... command+t... like in TextEdit). Same with colors (so no spot built in... I think you could probably jury-rig it, though).

    \n

    It has a very easy learning curve. If you're used to pro apps, you'll notice that things aren't where they're "supposed" to be, but you'll find them very quickly. As an example, rather than a Pathfinder Pallet, you have "Intersect" in the Objects menu. If you've used Fireworks, you'll be familiar with using that method as opposed to the full Pathfinder.

    \n

    Speaking of Fireworks, I'd say this would be a very worthy alternative to it. I wouldn't go so far as to say that it's an easy replacement to Illustrator, but for $79, it's really good. If I had $80 right now, it would be gone And that's after only 15 minutes

    \n

    The one thing that stands out, so far, that I can't seem to find is a "simplify path" function. But I have to say that the bezier tools are far easier to use than most... more intuitive. And the Brush tool auto-smoothes (but this can be bypassed by holding down the option key as you draw), so this missing functionality probably won't be too big of an issue.

    \n

    Definitely download the 30 day trial and give it a whirl.

    \n Bitmap pictures are like a giant sandbox. Each picture is made up of millions of grains, which can be moved around, erased, and changed. This is very flexible, but your sandbox is only as good as the number of grains per inch of your picture is. So if you have very few grains of sand per inch, you can't make the picture really big, because the information isn't there. The picture will get very fuzzy.

    Vector pictures are made out of infinitely elastic rubber bands. These rubber bands can be twisted and turned, made into shapes and colored. However, they are slightly less flexible that Bitmaps. Their big advantage is that once a vector graphic is drawn, it can be blown up to any size without any loss of sharpness or detail. This is because, at heart, vectors are defined by their curves and lines. Vector graphics are an expression of the math behind them, and because this is all defined in relation to one another, they are infinitely scalable.

    Does that make any sense? (I realize this is very simple. This might also be a good place to break off this conversation into another thread if we need to continue discussing it)

    \n I use a 4"x6" Wacom Intuos (don't get the Ghapphire if you can afford the Intuos). I first requested a stylus and tablet at the office about two and half years ago because I was getting hand, wrist, and arm pains from repetitive point-and-click mouse use. (Yes, I did use it that much.) I loved it so much, it made things so very much easier that before the first week was over, I ordered the same model for my home computer (they were on sale, I was lucky).

    It's the best computer-related purchase I ever made. It takes a few days to get used to it, but once you have you get enormously more control and, in my case, a much greater range of movement so that I no longer get pains from repetitive motion. I have fallen into the habit of tucking the stylus between two fingers while I type (like now), making it quick to switch back and forth.

    I can draw or "ink" directly, the secret being to zoom in enormously. Don't expect to draw at the same scale as on paper; you need to be "closer" to the line to get good, non-shaky lines. I was never able do digital art, no matter how I tried, until I got the tablet.

    Enjoy!
