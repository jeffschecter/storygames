\n I realise this is an impossibly broad question, but someone might be able to prompt me.

I need a game to run at Furnace, a lovely little UK convention, in October. The problem is, my list of games-I-want-to-run is empty. And I've run all my go-to games recently.

For example, I don't want to run Gumshoe (I ran a four-session marathon at the last convention and I'm Gumshoed out). I can't run Lacuna (because I ran that at Furnace last year).

What should I run? Ideally, I need someone to say: Graham, you talked about this game a while ago, why don't you run that? Or this game is an old favourite of yours, why don't you run that? Or...you like this game, so why don't you try this game?

The two games that I'm tempted by are:

1. Poison'd, but that's an ashcan and
2. Some sort of hard sci-fi game that must exist somewhere but I don't know about.

You are absolutely not allowed to mention your game in this thread, unless I've previously expressed an interest in it.

Graham

    \n Shock isn't really my sort of thing. I hear it's very good, but not really my sort of thing. Not sure why.

    Now, Paranoia's a good idea. I could do straight, Kafka-esque, conspiracy-laden Paranoia. The problem is that the system isn't all that good. It's all about the setting and colour. I'd quite like a good, tight, rules-driven indie game.

    Blue Planet looks good, but it's too expensive for me to get on spec. The idea of pulling something off the shelf is appealing. Tunnels and Trolls. Toon. Something like that.

    Now...something's pushing me towards InSpectres, too.

    Or Poison'd, set in space.

    Or that Grand Theft Auto Narnia crossover I always wanted to do.

    I'd really welcome other suggestions.
