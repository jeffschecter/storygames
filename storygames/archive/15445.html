      \n The Multiverse plugin allows the admin to add as many world maps as the server can handle. The vanilla game already has three: world, world_nether and world_the_end. You can then create custom portals to link all of the worlds together, or set up player teleport commands which might cost in-game currency to use. You can also have different difficulty, conduct rules or inventory management for the different maps, but it takes a fair amount of effort on the admin's part to set it all up properly. This is important to prevent items spawned on the creative maps from being brought over to survival or PvP maps. If that all seems like too much, just keep the same format and rules across all maps.

      The dynmap plugin generates a real-time map for all of the worlds and sticks them on a web page for you so everyone can see who is on, and where.
