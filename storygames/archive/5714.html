\n So following on from the wonderful thread on what software people use to make their games ... I'm considering a PDF game, and I would love some recommendations on software to use.

My requirements are as follows\n
    \n
  • The game will have text and large greyscale graphics. I probably need to produce two versions, one with lower res (72dpi) images for screen use and a higher res one (200dpi +) for printing
  • I will probably write the text in word ... though I could use a text editor without it killing me, provided I can preview it easily and control pagination easily
  • I am using Mac OSX 10.3 (which is a version too low for Pages)
  • I have no relevant software other than Word, Inkscape (which I don't know very well) and Photoshop. I can do CSS, HTML, ReStructured (if needed) and the like, but I prefer to lay out in WISIWIG
  • I don't have any budget for InDesign, Acrobat and the like
\nSo ... any advice? Anyone produced games in PDF with print and screen versions? If so what did you use to do it?

    \n Ryan, it depends what you mean by "considering a PDF game".

    If you were producing a quick, free game, or something for Game Chef, I'd be suggesting PDF995 for Word. Free add-on, creates PDFs. Easy.

    However, if you even thought of using that to produce any kind of professional game, I would not be your friend any more. If you want it to look halfway decent, do consider layout as a separate thing, and then all the software packages from the other thread come into play.

    \n Word does not retain images at any higher resolution than (at best) 200 pixels per inch. If you stick in a 600-ppi TIFF image, then you now effectively have a 200-ppi JPEG. If you are making screen-resolution knock-about PDFs then it's not a problem. But Word is not layout software so I wouldn't recommend it for that.

    Scribus needs Mac OS X (10.3.9), but is often recommended: http://www.scribus.net/
    You need this page.

    Willing to pay a little?
    What about SwiftPublisher? http://www.belightsoft.com/buy/swiftpublisher.php
    ...or...
    Ready, Set, Go!: http://www.diwan.com/ready/prsg.htm
