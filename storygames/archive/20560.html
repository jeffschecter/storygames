\n It was described as a role-playing game, but it seemed like it could have been described as a a creative writing exercise as well or instead.

You were meant to play it online. The game created a small wiki describing a setting which the players created together.

Every player created a page which began with 'A'. The listing had to link to an as-yet-uncreated page which began with 'B'.

When everyone had made their 'A' page, everyone chose a 'B' page to write. You couldn't choose a page which you yourself had linked to.

Every 'B' page had to link to a 'C' page, and so on through the alphabet.

I remember reading the rules online, but haven't been able to find them again.
