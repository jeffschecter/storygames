\n Occasionally people here talk about Flash stuff.

I've been working as a Flash programmer1,2 for years.

I'm not going to be doing that for the next year, but I'm thinking about doing stuff that I think is fun.

Here's a few examples of stuff I've done:

http://www.windowslive.no/ - dynamic content read from an XML file.
http://www.sandvika-storsenter.no/butikker - searchable floor plan for shopping centre

So if you need some Flash programming, get in touch with me. If it sounds like fun, I'll do it. If it doesn't sound like fun, I'm not doing it.



Note 1: Programmer - not designer. That means I can program animations, functionality, all sorts of goodies - but I can't make graphics worth sh*t; all that's done by designers.
Note 2: Flash programmer - not back-end/database programmer. That means I can talk to databases other people have programmed, but I'm not all that good at setting them up.
