\n On Mighty Thews is a roleplaying game for creating pulp fantasy short stories, in the vein of Howard, Moorcock, and Lieber. It's zero-prep, easy and quick to play, and it's a heck of a lot of fun.

The first edition of the game is free, and can be downloaded here, or you can read it in html here.

The first edition plays great, and I and others have had many hours of fun with it. But since it was released I've had a lot of feedback from various sources suggesting improvements, asking for clarifications, and identifying problems. I've also learnt a lot about game design in that time, and a little about layout.

I decided it was time to make a second edition of the game. I've done a heck of a lot of work on it, and I'm very happy with what's emerged. I've made a few changes, re-written the text extensively, and improved the readability of the document extensively. It's now a much, much better game. So much better, in fact, that I think it's worth paying money for. I've never charged for something I've written before, so I'd like to make sure it's absolutely right before I release it.

To that end, I've made a limited-edition beta version of the game. This includes all the improvements I've made up until this point, which is from empirical testing a fun and exciting game, but I want to make it better.

I'd like to find five people who can play the game once or more, and provide feedback based on that. The game takes about two hours to play, with two to four players (including a GM). If you think you can commit to that, whisper to me in this thread and I'll send you the document.

Thanks very much,

Simon
