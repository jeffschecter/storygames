\n Hey folks,

As some of you may know, I started a podcast called The Independent Insurgency. This biweekly show is an hour long (or so) and consists of interviews/discussions with game designers about the design, publication, and post-publication process of complete or nearly-complete games (first [or should I say, 1st?] episode notwithstanding). Furthermore, I only conduct interviews face-to-face; I don't use Skype because I don't like how it sounds and it fucks up the flow of conversation for me.

The problem is that I have only one interview banked, and that one is due out this weekend (when I go to JiffyCon in Boston). I'd really like to get a few more in the can. If you're within a few hours drive of central NJ and are willing to chat, let me know please. Especially if you're going to be at JiffyCon, we should definitely hook up. If you have any questions about the show I'd be happy to answer them.
