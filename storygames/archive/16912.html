\n So my gaming group has been indulging me in some story game goodness this summer, and the next entree I'm going to be serving up is Danger Patrol. When I first started having my visions of running this for a group, Pocket DP was not yet in existence. Now that it is, I'm wondering which version would be best to run for relative newcomers to story games. We've done Fiasco and Lady Blackbird together so far and both went swimmingly (in fact, Fiasco set us back a few sessions as they insisted on trying out other playbooks. LB might do the same as we may end up doing a sequel!). I feel like having the more traditional character sheets of DP Classic might calm the nerves of people who like a bit more crunch, but I don't know what sort of mechanics I'll be tossing aside if I don't use the shiny and new Pocket version. Thoughts, recommendations, warnings?
