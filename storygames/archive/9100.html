    \n There hasn't been a Little Game Chef before. This one was based on the Game Chef competitions that have run each year. Actually, Andy and me had the following conversation:

    Me: Could Game Chef be about immersion this year?
    Andy: Well, maybe, but I'm not thinking of running Game Chef until the fall.
    Me: Oh, can I steal the timeslot and run a Game Chef-like competition about immersion?
    Andy: OK then.

    So, yes, I gather there'll be a Game Chef proper in the autumn.

    Little Game Chef came about because I wanted to kick about the topic of immersion. I'd only run another one if I had another topic like that: something that we'd talked about and that would be interesting to design games around. It wouldn't be until next year, definitely.

    If you do start your own competition, could you avoid the words "Little Game Chef"? I feel proprietorial towards that name now (even though part of it comes from someone else). Call it something similar, but not exactly "Little Game Chef".

    Graham

    \n There are little design contests all the time. Ones in recent memory include:

    - Sight & Sound (Sept 2007, Jason Morningstar)
    - Make Game$ Fast (Oct 2007, Joe Murphy & Graham Walmsley)
    - Murderland (Oct 2008, me)

    Basically, if you think you can get people's attention and run a good contest, you post something and it goes from there, with people choosing to participate or not, as they will. The only regular, semi-official thing is (Iron) Game Chef, which has run once a year since 2002, first at the Forge and then on its own independent forum.
