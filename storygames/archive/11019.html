\n Dreamation has changed the way many of us run events. That's cool, it just means we need to do a little more planning before hand and UNITE AS A COMMUNITY (but let's not get too huggy, alright, I don't like the look of some of you). I figure it'll be handy to have a thread where we can say "I'm interested in running a session of this 4 person game" and then others can chime in and join the crowd.

So what do you want to run but need help with? Let us know.

NOTE: I realize there's the ige wiki, but i think this kind of discussion would clog it up needlessly. Also this is a place more folks check regularly. If i'm wrong and the wiki is a better fit, say so and we can close/move this thread, no biggie.

    \n So I'm going to run Empire of Dust. It's a great game that i don't get to play enough. It even runs really well with a big group, believe it or not. So if I can't find another GM who's interested in it I'll just throw down for a big group of like 7 players.

    But i would rather split the party into two teams of 4 each.

    See, i was thinking about it and this new event format enables some cool parallel storytelling techniques, like...

    \u2022 having two or three groups meet up and then separate
    \u2022 Player swaps between groups
    \u2022 two groups knowing or unknowingly competing for the same goal
    \u2022 GM swapping

    I want to try some of these tools out, anyone interested in joining me?

    Things to know about Empire of Dust:
    \u2022 the game is super easy to both teach and learn
    \u2022 it's really fast paced action
    \u2022 players really respond to it
    \u2022 I will bring all the materials both groups need to play (I've got a double set of tiles and can run the rules pdf off my laptop while another group references the book)
