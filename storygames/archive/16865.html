    I played around with TiddlyWiki in the past. It's neat, self contained, and you can share it like a file.
    This looks good. How does it look in file format? Does it combine into a single document or is it forever presented in wiki format?
    \nIf by "file format" you mean the underlying format of the wiki, it's essentially a series of div elements containing each entry (or Tiddler). The quick and easy way of making a single document is to open the selected tiddlers you want in the doc, then use the print function. You can then use one of the many print-to-pdf options out there to make a PDF.

    If you're familiar with someone who knows a bit of scripting, it should be easy enough to make a script that parses the HTML file, pulls out all the tiddlers (or a selected list), and outputs them to some document.

    \n Summary so far, for anyone watching:

    TiddlyWiki was close, but it was missing functionality. It doesn't have natural support for visitor comments, though there is a plugin available. As Alexander has advised, it would be possible to get TiddlyWiki to export the pages with some scripting, but it's not a feature of the software.

    Google Drive supports comments and exporting. It's the most appropriate suggestion so far, but it lacks structure support. If I want many pages covering many topics, each accessible through a hyperlinked list of contents, I've got to create the hyperlinked list of contents. Maybe I'll have to live with that.

    So, still looking for suggestions. Particularly, I want wiki-like editing, revision support with an export button and visitor marginalia. Any more suggestions? What are other people using to develop their projects in public?
