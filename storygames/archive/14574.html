    \n Well, IAWA is made for long-term play. It just is. Playing it as a one-shot with only one chapter is pretty much a waste of time. I mean it's fun, but it's just a bidding system and a random monster, I mean situation, table. The owe list is what makes this game shine and it only works with long term play.

    The problem with sticking with the same set of characters is that the best interests that drive the game are best if the characters are working against each other, at cross purposes, or in some other way fighting each other. And if you're committed to the same set of characters, then nobody can murder each other or even really decisively achieve or not achieve what they need to. IAWA has a strong element of PvP, which is not a good match for the you-and-me-against-the-world traditional structure.

    I have done this with In A Wicked Age. It worked great. We did this:

    \n
      \n

    1. Create characters and a starting situation. Decide how the PCs know each other. Assign stats and 1 particular strength just like in IAWA. We basically used Sorcerer as our guidebook for character creation for this.

      2. \n

    2. Write each PC's name 3 times on the We Owe list.

      3. \n

    3. The first "oracle draw" includes 1 card crafted specifically around the PCs, plus 3 random draws.

      4. \n
    \n

    Play IAWA as normal, with the expectation that players are to cross their names off if they're not implied by future oracles.
