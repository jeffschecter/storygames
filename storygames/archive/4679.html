\n So there's a new site coming soon, I read about it today at if:book, here:

Unbound Reader

I'm not super clear on how it's going to work (still waiting for my beta account), but it seems like you can do live chat while you're reading a text. It also supports chat within certain bounds of the text, like if you only want to chat about the first chapter, but not the end, the site will only include users reading the first chapter like yourself.

Anyway, while I'm not sure if I like the idea of chatting about something I'm concurrently reading, this does seem like a cool platform for a chat-based story game. The shared text could introduce content and mechanics for the players to riff off of? Just wanted to throw this out there.
