    \n The playbooks are the same as DW, however NA has:
    Racial playbooks for:
    1. Small & Sneaky races like goblins
    2. Marauding Horde races like orcs
    3. Towering Brute races like ogres
    4. Lycanthrope races like werewolves
    5. Restless dead races like skeletons
    6. Hungry dead races like vampires
    Each of these racial playbooks include:
    - Things to replace choices on the standard character sheets: name, look and an alternative racial move.
    - Possible true facts about your race so you can help define it.
    - Some relevant Compendium Classes. Generally these will be suggestions relevant to the particular race that any race could take. But sometimes they require particular races.
    - If relevant, details of how this race can connect to a Last Breath move.
