    \n I usually manage to bolt minis play onto whatever game I'm running, because I have lots and want to use them.

    YEARS ago, I bolted a concept like mana onto D&D, before I'd read the Gygax Dragon article. (I will concede that it's possible I HEARD about mana in an off-hand conversation beforehand, though.)     AH, wait.... the OP is about what SYSTEM onto which we bolt OTHER things. Well, hell... any of them that support--even in a minor or orthogonal way--using minis, I guess?

    \n Games I can run just about anything with: OVA, BESM, TSOY/Solar, Story System, Over The Edge,PDQ/PDQ#, Risus, and FATE/Awesome Adventures. Not in that order.

    I really dig a good generic system that is small, light, and treats setting elements (like equipment, organizations, etc) as add-on aspects that aid the character who uses them. Effects-based power creation is another plus (just handle the mechanical effect and the table will decide how it looks/works). This lets the table be creative, not just creatively use the setting bits designed by some designer somewhere in a high tower.

    I hope to add the following to my toolkit for just these reasons above: Dr. Who, Wordplay. Just have to get time to read them and let them gel.
