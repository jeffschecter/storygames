    If you're interested in voice-based games using a virtual tabletop, check out Infrno.
    \nHow's setup of the client? Basic unzip or setup.exe, no dependencies?

    \nThe virtual tabletop is flash-based. It's too limited for miniatures based combat, but is more than adequate for sharing images and text.

    It works very well for most indy-games, but it's missing support for cards. It's not the best table around, but it comes with the rest of the site that provides scheduling, forums, game management, and chat. The community is what keeps me going back to infrno, not the technology.

    One warning, the version of flash used in google chrome can behave weirdly with the Infrno Table, so you might want to try it in Firefox.

    Most people do not use the integrated audio-video system, we use Skype instead. The sound quality is incomparably higher.
