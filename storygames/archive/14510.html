\n I've started running a play by email AD&D 2e game set in the Forgotten Realms, I've mostly been ad-libbing it since doing rolls would slow things down, and so far so good. When I did do rolls for the players though, I found an advantage of handling all the rolls for a round first, and then interpreting the results, instead of doing it one by one. I'm getting some events happening that I normally wouldn't in a face to face game.

What are some other nice perks about playing a PbE game that aren't present at the table that I should look out for and perhaps take advantage of?

    \n The only thing I've ever liked about PbP (never done play by email) is that you've got time to think through what you're saying so you get a more prosaic quality to your fiction. Everything's well thought out and well written, unlike playing live in person.
