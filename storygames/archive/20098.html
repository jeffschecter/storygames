\n Hi, so I'm working on a setting that emulates Dragonball Z and other Japanese media. This is a more complete, trimmed version of the Wanderer System rules. I've still got to finish the Trap/Hazard section, social interactions, add more Techniques, fill in the Quest Section and add the personal Story Arc section, plus roster sheets. I posted this version online because I had some people intrigued enough to actually want to switch their upcoming game to the system.

If anyone has any thoughts I'd love to read them!
