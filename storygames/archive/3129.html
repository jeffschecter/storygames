\nI don't know if I've mentioned this here before, but artpad is a shockingly handy little tool for long-distance doodling. The one thing it can't do, at least so far as I've found, is allowing others to edit an existing work. Still, it's handy for quickly getting across distinctly visual information. The above link is from a recent discussion with Chris Chinn about board mechanics for Do.
