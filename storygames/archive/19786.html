    \n Ghost/Echo is itself a hack of Vince Baker's Otherkind with some bits thrown in (AW style Moves, for example). I don't know of other games directly derived from Ghost/Echo, but I know of several other games and hacks in the same family, that also use Otherkind's core dice mechanic as their basis. Games like Psi*Run and Mobile Frame Zero to start with. (And I wrote a couple of games that work along those lines.)

    \nHERE is a pdf of the pre-gen characters Orlando_Wilson made. I think only jpgs of them were posted in the original G/E thread.

    Steampunk Crescendo is a vampire-y steampunk game based on Otherkind Dice.

    FUBAR, by Michael Wenman is Otherkind Dice based (or was in the first version, I'm pretty sure).

    If I remember right, Tony Dowler's alt-Renaissance game Principia uses Otherkind Dice rules.

    I wrote a G/E hack for fantasy thieves in a fantasy city called Thieves Can Too, Motherfucker! and did a significant revision of it later, renamed Den of Thieves.

    No guarantees on any of this stuff, I'm mostly working on old memories here.
