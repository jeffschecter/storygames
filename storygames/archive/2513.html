\n So, the Wiki still isn't fixed (I get an error when I try to edit) so I thought I'd start food discussion over here. What we have so far:
Breakfast:
--Bagels and cream cheese (lox and tomato and onion if you want)
--Pancakes

Lunch/Dinner:
--Spaghetti (w/meatballs on the side)
--Foil dinners (what are these, exactly?)
--Sandwiches
--Falafels
--Fajitas/Burritos
--Soup and grilled cheese/other sandwich
--BBQ stuff (meat, veggies, potatoes) if there is a grill available

Dessert:
--S'mores!
--Rice krispy squares

Please give other suggestions! These are technically enough to do one for each meal, but it's silly if you don't like what's being offered. Personally, I'm not a fan of sandwiches. Grilled cheese is okay, but sandwiches are icky. Thus, I have added falafels.

It's actually much easier if we share food, because then we're not competing for campfire cooking time, we only dirty one set of dishes, and if you cook, someone else has to clean up!

Discussion: Begin!

--Nancy (Food Maven)
