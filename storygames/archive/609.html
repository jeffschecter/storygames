    Okay, I can make that happen with my "deck of cards" code and I can set up a MUSH on my server just for playing WGP. I'm happy to program other reasonable dice systems and stuff, too, if people want to use the space to play other games.

    If you don't know what MUSH is, it's like IRC but with a virtual place. Where IRC has handles, MUSH has characters that you log into. Where IRC has channels, MUSH has rooms and they're connected by doorways and you can make your character walk around and talk to other characters in the rooms you visit. MUSH also lets any player create objects with descriptions (great for recording notes or making text props), build rooms (if you want a set of rooms for different gaming purposes, or even a LARP-like set), and code in a terrible but fairly complex scripting language. I'm really really good at that language, so if you want something, let me know.

    I think it'd be cool to have an indie rpg hangout-and-chat-and-game MUSH. I'd call it Foundry, as a counterpart to Forge. Or something equally unclever.
