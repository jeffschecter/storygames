\n Roguelike is a term used to refer to crpgs that share particular features of the 1980 crpg Rogue (stuff like turn-based gameplay, permanent death and randomization).
(Note that Rogue wasn't the first game to use all these features, but it popularised its combination.)


It made me wonder, what tabletop rpgs do we have that could name a subgenre of the hobby and what features would be shared with games considered ____like?

    \n Even though the official abbreviation is "PbtA", a lot of AW hacks wind up being called "*World games", which I think is exactly this sort of thing. *World games use the 10+/7-9/6- style of player-facing dice rolls, sometimes with a tweak (that makes it look slightly different, but it's the same core), and almost always with character-exclusive "playbooks".

    D&D-likes can be defined, I think, by level-based advancement (often with feats), a game system which includes explicit combat statistics like HP, armor class, and attack bonuses, and definitely by the existence of a class system.
