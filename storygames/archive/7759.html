\n So, what are people's favorite system's for trying new things? Not sure if this has been done before, but It's a topic that is coming up in some of my games.
\n
\nI actually really like using D+D(3.x or 4e) as my testbed for any sort of scene framing exercises. I think that everyone in my group has a really good baseline of how much fun and interest D+D is supposed to generate. So if we change up scene framing, or even world concepts, we can pretty accurately measure what it did to the game.
\n
\nI really like using Buffy to test prop stuff. I think I would also like using PTA, but I don't own the system. Using a TV-based game to test props, sound effects, theme songs, and even background music just makes tons of sense!
\n
\nSo how about it? Got any favorite games to test new ideas

    \n The utilitarian system I return to again and again is Solar System. Hacking/tweaking Keys, Secrets and Abilities lets us tune it for a pretty wide range of effects. It's easy to try out some new technique to see how it works, and since we play all the time, we know right away if it an improvement.
