    \n You need to have some seriously codified prep for this to work effectively, I think. Otherwise the new group will spend so much time interpreting the prep document that they will have effectively done a new set of prep. So for games like Shock: this seems like it would work pretty well, because most of the interpretation can happen once the game starts; but something like Sorceror I am not so sure.
