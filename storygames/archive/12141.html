    \n Actually, I'm interested in that, too. It's a very similar situation to the one Primetime Adventures had between the first and second edition: the improvements I could see for the game are so niggling that I can't imagine the game really changing much. It's a very honest and streamlined thing that does exactly what it purports to. I could see some rewriting of the advice to sharpen it up to modern standards (the text suffers just lightly of '90s-style inexact generalizations in places where it should talk straight about the methodology), but aside from that I don't really know what would need to be changed in the system. That's a rare thing.

    \n Hey guys -

    I was about to flag up that post, but Kev beat me to it.

    In short, the game itself has barely changed - I've tweaked it slightly so you can lose survival points for checks other than combat, I've poked at how monsters work mechanically and I've introduced Bad Habits, which are personalised cliches that characters can have.

    Where all the changes and, really, additions have occurred are in the much-expanded monster chapter (with just as many monsters, but also a bigger look at designing your own), a new chapter about running the game in any horror genre, and a sizeable GM chapter with lots of advice and techniques for running horror games and, specifically, games of Dead of Night.

    So that's your lot - if you're only going to check out one edition, I'd recommend waiting a few weeks for second ed. It still has all the stuff that made 1st ed great (apart from the really small form factor, which is hotly debated whether it was great or not anyway), plus loads more useful content.

    Cheers

    Andrew
