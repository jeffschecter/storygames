\n I've finished the first alpha version of the GM-less (solo or coop) package for Anima Prime, my Final Fantasy and anime-inspired action RPG. I'm going to be trying it out a lot more over here, but I figured if any of you out there would like to give it a try as well, that'd be great. You can play it alone or with several players; I'm not sure I'd suggest more than three players right now. You need to be familiar with the Anima Prime rules, but the complete game is downloadable from the website for free, and it's not all that complicated.

Anima Prime: Unity Core is a card-based system. There are:

1) scenario cards showing win and lose conditions, the sets of other cards to use, a character scene track, and special rules;

2) adversity cards that randomize whom you're fighting; and

3) circumstance cards that make fights more interesting, including features (fighting in a thunderstorm or in narrow corridors), goals (catch an intel runner or save some civilians during the fight), and hazards (minefields, collapsing buildings...).

The adversity is automated via action tables. If you've played any of the DnD board games (Castle Ravenloft, Wrath of Ashardalon...), you'll be familiar with the basic idea of going down a list of actions and picking the first one whose requirements are met.

Actually, the circumstance cards can be used for regular Anima Prime games as well (to add some spice to your conflicts), as can the action tables (to teach new GMs how to run adversity).

It's labeled as an alpha because it only contains half the cards and adversity that I'd expect a full version to have.

Oh: some assembly required (cutting out and, preferably, sleeving 40 cards).
