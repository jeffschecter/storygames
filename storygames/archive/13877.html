\n I got into a dumb discussion about "what was the first role-playing game to" a few days ago. I figured I might as well immortalize the useless geekery here.

The idea is to build a list of role-playing innovation (I'm mostly interested in mechanical innovation, but whatever). In the post that follows this one, I've suggested some of the "firsts" that came up in this discussion (and with some later Googling). Some or all of these are probably wrong. If you know of an earlier example, mention it here. Also, if you are interested in some other "first", post it with your best guess as to what game can make the boast, and others can post earlier examples if they can. Make sure you use only official release dates. Betas and ashcans and play-tests and such don't count.

I will try to keep this first post edited with the latest "winners" mentioned (for as long as I can stand it, anyway).

Edit: results so far\u2026\n
  • First role-playing game to support solitaire play: Tunnels & Trolls, 1975
  • First role-playing game to use a summed dice pool for resolution: Tunnels & Trolls, 1975
  • First role-playing game to use a skill system: Bunnies & Burrows, 1976
  • First role-playing game to use life paths: Traveller, 1977
  • First role-playing game to use a point buy system for character generation: Melee, 1977
  • First role-playing game to offer a universal resolution mechanic: The Fantasy Trip - In the Labyrinth, 1980
  • First role-playing game to use plot points: Top Secret, 1980 (called "luck points")
  • First role-playing game to use player's actions mechanically: Toon, 1984 (making players laugh forces them to skip a turn)
  • First role-playing game to weave character personality into mechanics: Pendragon, 1985
  • First role-playing game to avoid using/tracking money: DC Heroes, 1985
  • First role-playing game to run GM-less: The Bullwinkle & Rocky Role-Playing Party Game, 1987
  • First role-playing game to use archetypes to speed up point-buy systems: Star Wars 1st edition, 1987
  • First role-playing game to use successes from dice pools for resolution: Shadowrun, 1989
  • First role-playing game to use card-based resolution: Lace & Steel, 1989
  • First role-playing game to give players un-vetoable narrative powers: Impro No, 1990
  • First role-playing game to make everyone the GM: Impro *, 1990
  • First role-playing game to eliminate character ownership: Impro *, 1990
  • First role-playing game to make world-building a part of game play: Impro *, 1990
  • First role-playing game to allow players to invent their own traits: Impro *, 1990
  • First role-playing game to use karma resolution: Amber, 1991
  • First role-playing game to have players take the role of actors playing the roles of the characters: On Stage!, 1995
  • First role-playing game to auction the right to set the scene: On Stage!, 1995
  • First role-playing game to use formal endgame mechanics: Puppetland, 1999
  • First role-playing game to use social contract as explicit game mechanic: Universalis, 2002
  • First role-playing game to provide mechanical rewards for entering conflict: Universalis, 2002
  • First role-playing game to give relationships mechanical weight: Universalis, 2002
  • First role-playing game to use keys: The Shadow of Yesterday, 2004
  • First role-playing game to use a win/win mechanic: \u2026In Spacaace!, 2005
  • First role-playing game to heavily rely on random generation of starting situation: In A Wicked Age\u2026, 2007
  • First role-playing game to leave out a published list of equipment: ???

      \n Braunstein was definitely a role-playing game, by all our normal definitions of the idea: people had their own characters, they talked to each other in-character, they decided what their character would do based on their character's personality (q.v. cocky student A from Braunstein I), etc.

      Wargames had referees, which were different in many regards from GMs. I'd argue that a referee just judges the participants' actions by the rules (usually moderating PvP), while a GM speaks for and can shape the fictional world. I don't think you'd see a wargame ref describing the action of NPCs, if you see what I mean, and you don't turn to the ref to paint the fictional world or spontaneous insert facts into the scenario.

      Anyway, assorted cans of worms, probably not for here and now.
