\n As said in the thread about good maps for games I think a collection of "blank label" maps would be a very useful tool, especially when coupled with some kind of game to help the group define the world of that map in a fun way.

Also, it would have to be very short and simple, max 2 pages I'd say, so that it could be included with the maps.

Inspired by Fred's new Faith Faces and Fingerprints campaign world creation I was thinking about a Token/Resource spending mechanic that would allow a player to state a fact about the world.

For example:\n
    \n
  • Each player starts with N Chips
    • \n
  • Player points to a place on the map, declaring that the new fact will be about that spot
    • \n
  • Each player bids one or more chips (secretly) to gain the right to estabilish a fact
    • \n
  • The player with highest number of chips has the right to write a name in a label and estabilish a fact about that place
    • \n
  • The player that "won" spends his chips, the others keep them.
    • \n
\nThe turn goes on.
This model would allow a kind of competitive building where if one player is trying to estabilish facts on another player's chosen place he can be stopped by bidding higher. It would probably lead to a kind of ownership feeling from players ("This is my kingdom of Kalaya").

...Or, the collaborative version (this one shamelessly stolen from Fred):\n
    \n
  • Each player starts with N chips
    • \n
  • The first player asks a question about the map (place, label, whatever). He's the only one that cannot answer the question.
    • \n
  • One of the other players pays one chip, and answers the question.
    • \n
  • The player that answered last, pays one chip and asks another question.
    • \n
  • ...and so on.
    • \n
\nIn this case the N would be much lower I guess.

Both procedures, I think, need some more spice added to them... so, what would you change/do?
