    Anyone have any experience playing The Play's the Thing? What's it like?
    \nI have gone through it and am preparing to run it, but haven't run it yet.

    It's tabletop rather than larp. You can stand up when doing scenes, but it's written with a table in mind. There is a GM (playwright) and players. The GM defines a plot, but the players can make edits - which they may have to roll dice for if the edit is contested. Each actor will have a collection of Parts, Plots, and Props that they can bring into scenes in different combinations - which have compels and bonuses they give.

    It's very rules-light for a tabletop RPG, but I feel like it is rules-heavy for improvising Shakespearean scenes.
