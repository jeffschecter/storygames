    Posted By: thorAndy,

    My home system is giving me the can't get through message. But I don't have an IT guy to push the buttons. Should I just power off my modem or do I need to call AT&T?
    That's a good question:

    1) If you have a modem-router combo, powering it off and back on wouldn't really hurt.
    2) But if it doesn't work, then yeah you'll be sending AT&T an email or call (tell them it's happening with "a bunch of sites" :-)

    Here's my workaround I sent to Jason. It assumes that you're also at work or somewhere else that can see the site:

    ==========================
    ==========================
    So I'm thinking that some DNS servers that serve some sites at your
    work are updating, but the one close to you
    isn't.

    If you are running Win you can do this which maybe will alleviate the issue (though you will
    probably want to give the info below to IT, telling them it's
    happening with "several servers", cause there may be a genuine issue).

    From your broked PC:
    (go to command line or start->run->"cmd")
    > nslookup www.story-games.com
    Write down all that info. It's telling you the (bad) IP for story
    games, but it's also giving you the DNS server IP or name that it's
    checking against. That DNS server IP and name is what we want to remember (when we call IT).

    Then do the same from a "good" PC. Compare the two. I'll bet that the
    DNS server doing the lookup is different.

    You can give this info to IT, and tell them that this DNS server
    (broke) needs to be "flushed".
    --------------------
    (note to Thor: This last part won't work if you're on seperate networks, this is more for people who are at one company, where like their PC doesn't work, but the one "downstairs" does or whatever)

    Or, another trick: Write down the IP address of that "good" DNS
    server. Then try to ping it from the command line. If it responds,
    then we're good (if not, then it may be part of another network and
    can't do this trick).

    Then Right-click on My Network Places, go to Properties, then right
    click on "Local Area Connection", and again go to Properties,
    Double-click on "TCP/IP" to bring up a new window.

    On the General tab, where it says "Obtain DNS server address
    automatically", instead choose the other option, and manually punch in
    the DNS server IP address used in that second (good) nslookup. Then
    click OK, OK, OK, etc That should work temporarily.
    ================================
    ================================

    DaveC just wrote the new IP address into the Windows Hosts file, so that's another way. Windows grabs any entry in the Hosts file and circumvents even asking a DNS server.

    The Hosts file is a text file in C:\WINDOWS\SYSTEM32\DRIVERS\ETC (you might not have one yet, so just make a new text file and call it HOSTS (no three letter extension!)). That's the default directory for Windows XP, at least. Edit this file with notepad and enter the IP address in the following format:

    64.111.120.140 www.story-games.com

    That's it!
