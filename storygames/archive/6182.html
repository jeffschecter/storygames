\n I've just had a revelation.

Red Box Hack is a targeted game that's easy to read and has kewl powerz. Wicked Age tells you to make them up yourself, and pretends it's easy. You can only use one wicked age power at a time but if you're smart in Red box Hack you can use lots in the same fight.

That's right - IAWA tells you to be dumb.
