\n So I was watching a Manchester United match yesterday and it got me thinking again about this idea I've had to run a football-centric game in the vein of Dream Team, only focusing instead not on a fictional Premiership team, but a more modest team in Football League One. There's plenty of drama to be had both on and off the pitch, though, so it seems like a natural for a group of people wanting to play out some soapy melodrama punctuated by on-field action.

But what system to use? I'll admit I considered PTA because it seems like every time I think of some idea that would make for a good game, inevitably someone will come along and say, "PTA can do that." But I'm unsure of the game's ability to deliver the kind of dramatic match play I envision, where players put it on the line for success (or dramatic failure).

Which brings me to Up Go the Heads, a dirt-simple system from Gregor Hutton that seems at first blush to do everything I want for this potential game. But I know that it never hurts to get feedback from people who've actually played the system in question.

Something tells me this is going to be a pretty select group, if it exists at all, but if you've played the game: how did it go? What did you do? Would you recommend it for the kind of game I'm thinking of?
