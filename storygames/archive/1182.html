OK, as Ben pointed out, there's some "Kerfluffle". I love that word.

There's a lot of solid, grounded, discussion going around on people's blogs. Some people are discussing lightly, others are discussing with meaty word-fists, but it's all respectful, focused and progressive.

BUT: OK, the Forge Diaspora Strikes!!! Oh Noes!
Discussion is here, there, everywhere; everywhere but the Forge! What are we to do???

So in this thread I'm going to do my civic duty by linking to all posts in the Great Kerfluffle of 2006 and posting them here, so that everyone can read up and be in the know. I'll update (reedit) this post daily. Please add links and their context in this thread as posts, I will take those, edit this first post and add them here.

I'm going to try to keep them organized by topic, then by chronology so that they can be read (replayed) in context. Make sure you catch the comments, because that's where a lot of the creative juice and ideas are.

This is not the thread to talk about the Kerfluffle, just to group links about it.

Here's the format I want:
1) URL
2) Context ("Raggedy Andy talks about how Polaris sucks")
3) Date the thread/post was originally made (if possible)
