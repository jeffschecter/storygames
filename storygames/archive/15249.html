\n Hi all! I have this small game system I'm working on, and I'd like to show it to some fellow players.
\nIt's a generic system I've temporarily named "Spiral;" it revolves around the consequences of the use of power and loss of "humanity." Playsets can be built around the very simple game rules to bend the game towards the desired feeling and results.
\nYou can see it here: http://www.box.net/shared/pga9q64b3j32i6r4x9oo This .pdf contains the rulebook, some sample playsets and a very rough char sheet.
\nHave fun!
