\n Dreamation 2015 is going to be an amazing gaming convention in Morristown, New Jersey, scheduled for February 19-February 22, 2015, hosted by Double Exposure.

Once again, Kat Miller and I are once again organizing an Indie Games eXplosion. If you would like to run games as part of the eXplosion, please fill out this Google form: http://goo.gl/forms/jceJAdkPCQ

The deadline for events is January 4, 2015.

We only schedule tabletop events through the IGX. For LARPs and LARP-like events, please contact Double Exposure directly as soon as possible, as space goes quickly!

If you have questions, don't hesitate to contact us either on Google Plus, or via Gmail: indiegamesexplosion

As events come in, I'll put together a preliminary schedule so we can watch the awesomeness grow before our very eyes!

If you have already submitted events directly though the Double Exposure website and would like to be considered part of IGX, please fill out the IGX form with the exact same information you submitted, so we can include you.

Thanks, and please pass this along to anyone I may have missed.

See you in February!
