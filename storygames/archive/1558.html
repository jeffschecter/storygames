    Most D&D variants that include some sort of fortune point take a page form the handling of Force points, and make them a scalable number of d6 (such as 1d6 for level 1-5, 2d6 for levels 6-10 and so on) which are added to a d20 roll. Alternatives include rolling an additional d20 and keeping the better one, or simply allowing a reroll.

    The sole point I would raise is the same issue with 7th Seas' Drama dice, which you could roll as bonuses _or_ keep as XP. The problem was that it was cool and fun to spend them, but it was smarter to hoard them. That resulted in less fun, so I would either decouple them from XP or change it so spent tokens translate into group XP. As an example, for every X tokens (where X equals the number of players) spent, grant the group one standard reward (the XP for an encounter with a CR of their level). This means that _everyone_ benefits from the funniest guy at the table.

    Additionally, if you want something a little less PTA but still dramatic, Chris Aylott's "Dynasties & Demagogues" includes feats which are rather like TSOY's keys (though some are even more meta). When a situation comes up, or the character (or player) does something in particular, they get an action point. The model is limited, as it was written to not upset the apple cart, but if a player of GM is comfortable with the idea, it's easily extendable.

    -Rob D.

    Posted By: Rob DonoghueMost D&D variants that include some sort of fortune point take a page form the handling of Force points
    I first saw it in the d20 AEG supplement 'Good' as Action Dice. AEG later also used it in a more refined form in Spycraft. Is Star Wars the original version of this for d20? They were 'exploding' dice that everyone got only so many of, but issued at the beginning (or by GM/group concensus when something amazing happened), by the GM, with a couple of different ways to spend them - roll to add the result to an attack or skill roll, spend n dice to auto-confirm a critical threat roll, etc.

    We used them in that form in my most recent long-running Dnd3.5 game. One player described them as 'like cheating, but better!'. We didn't have any 'cash-in' rule, it was use them or lose them, which helped pacing - everybody made sure they had a few to use in the big fight and almost all of them got used every session to do awesome stunts that wouldn't have been attempted otherwise.

    So yeah, use'm, love'm.
