Making crazy building falling down sounds as I did so.

So, there is clearly some balance issues involved if the townsfolk are to stand a chance at winning, so here is a solution that we haven't tested yet, but should help.

The Building's Strength

10 of the playing cards are assigned a die, which can be used once by the player holding the card. Split the following dice among the ten cards: d20, d12, d12, d8, d8, d8, d6, d6, d6, d6.

2 of the playing cards are assigned the role of enter and sticky. On a sticky card, all damage that the Monster does doesn't work this turn. And enter is used with the boardgame portion of the game (see below).

The game needs some tweaking, but each game takes about 30 minutes, and it will probably turn out to be a kind-of-fun if not fun board game.

Optional: Board Game Rules
  • The cards become hex tiles that can be laid out however the townsfolk want. Except each hex must be touched by another hex on two side minimum.
  • The monster enters on the Enter Tile.
  • Max of five townsfolk on the board.
  • Monster can only destroy tile that it is on.
  • If Monster loses combat, townsfolk can push it back onto a destoryed tile.
  • Monster can only move onto square that either: a)previously destoryed or b)where townsfolk was just killed.
  • Townsfolk can move one hex per turn and then attack. They start where ever the players want.
  • Only two fires can be on the board at a time. If the Monster butts up against fire, that fire's score applies in all combat.
  • Monster can only move through fire if there is no other direction to move. One die level damage and gain one Anger die per rules.
If The Building's Strength Rule Don't Work To Almost Even Up the Game
Fight, Step One: Roll d6 per rules. Characters with 1-2, 3-4, 5-6 roll together. Or characters roll odd or evens roll together.

    Larry, I actually can't believe that I played this. Thanks for the idea.

    The game is still pretty rough and doesn't play fair as written in the rules. Ha!!!! The monster should win!! I've played five or six solo versions of the game and am getting closer to something that might be pretty fun It's so hard to tell solo because there is a lot to keep track of.

    These are the changes that are getting closer in the board game:

    1. Townsfolk dice = d6, d4. These increase in die size on a win and now do not add additional dice.
    2. IKWeapon = As the IKW stat increases it stays with the character even if the character gets squashed. It can be rolled once it is in box three on the character sheet.
    3. NOT TESTED YET: Add d20 hex, remove blank sticky hex.
    4. Hex damage only occurs on the 1st round it is occupied by the monster, but all attacks during this round can use the extra die.
    5. When character squashed, monster player can push character back three hexes. But, if townsfolk are unable to attack during a round because no character is near the monster, one may be moved near the monster to attack.
    6. NOT TESTED YET: After Monster is totally roped down, townsfolk can push the monster back into destroyed hex when they win, and that hex is undestroyed.
    7. Now playing with three townsfolk max. Four may work, though.
    8. Monster can move: 1) Over any vacated spot that a character was squashed in. 2) Anywhere if no townsfolk around.
    9. When monster moves from a hex it is destroyed.

    I'll probably do a couple of solo games tonight if I don't do anything. I haven't played with the Fire Rules on the recent playtests, but I will tonight. I expect with the rules above combined with those in the manual for the board game and the fire rules, this should balance out pretty nicely. Here's the story arc that I'm shooting for with the game:

    • Monster beats the hell out of the villagers.
    • Villagers get stronger with IKW.
    • Monster almost destroys whole town.
    • Villagers push monster back.
    • Monster gets stronger via Anger.
    • Monster and Villagers very close in strength.
    • Final battles could go either way.
    • A little luck could tilt the game either way early.

    After I get this thing close to balanced (with the fire rules, it may be there), I need to make some sort of incentive for the different players playing the townsfolk. Not sure what yet. Maybe it doesn't need it.
