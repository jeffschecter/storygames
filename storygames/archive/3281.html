    \n Since you can never have too many charsheets, here's mine:

    character sheet
    npc sheet

    Based on the concepts that:
    a) Grouping skills by default pool rather than by "level"--the standard sheet does the latter, and this seems to me to be a good way to end up with way too much erasing and rewriting when improving stuff?
    b) Using chips to keep track of pools, so no need for checkboxes.
    c) I personally prefer "free" spaces to write stuff down, with no lines. (Likewise I always use unlined notepads, et cetera.)

    I liked the little rules summaries at the bottom of Jason's sheet, so maybe I'll steal that. Not like attributes/keys/secrets need the amount of space they're getting.

    (Like Jason, we print a separate strip with full secret/key details, updated as they change--three players, so I print three columns, each of which fits one character's stuff pretty much just right, cut 'em up and the players can stick 'em to the side of the character sheet.)

    \n I have updated my TSOY character sheet.

    Changes:
    1) Moved Keys and Harm tracker to the left side so that you can paper clip the harm tracker if you like (although thinking about that now, it doesn't really serve a purpose.
    2) Shrank the Pool bubbles so there are now 13 of them.
    3) Added slightly more explanation to the key buyoff advance row.
    4) Added a reference page column to each of the 3 categories.
    5) Added a Buy Reference to each row.

    How the Buy Reference works: When you purchase an advance, instead of filling the in the square, write the Buy Reference there. This way you see exactly how your character advanced and changed.

    I hope everyone enjoys it.

    Oh - if anyone actually uses it. I would love it if you would scan in the completed sheet so that we can see how your progressed. I think that would be awesome.
