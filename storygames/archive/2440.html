    While I've never tried explicitly playing MC GMless, I believe it can work with no rules tweaking, based on my last experience of playing it.

    I GMed a game of MC at Conception, and found myself with six players. Given that we had to create the world, I quickly realised that there was going to be no way of running a satisfying game for that many players in four hours if we were also going to take the time to create a bunch of NPC antagonists. The solution we went for was to limit significant characters to PCs, and to create a relationship map that would define alliances, emnities, family relationships, etc. between all the player characters. We then started off with the joint kicker of a mysterious magical murder committed by one of the PCs (we didn't know which one at this stage) and ran from there.

    Once we started actually playing all I did was explain rules, frame scenes and do some hand-holding during conflicts. After the first hour, once the game had gathered some momentum and the players knew what they were doing, all I did was watch and throw in the occassional suggestion; the scenes all came about organically, so there was no need for me to frame them. Group consensus about the appropriate use of power and magic tokens seemed to work fine, and I was never called on to do more than offer possible prices for magical definitions when the players were stuck. By the end of the game the players had taken over so much that I had to ask them to explain the conclusion to me, as I had no idea what was going on!

    Based on this, I'd say if you can front-load the conflicts, create strong relationships between the characters, limit the need for NPCs and make sure everyone knows the rules, the game will pretty much run itself.

    One possibility would be to borrow from games like Polaris and Shock: and to have the person sitting to the left (or opposite, or whatever) of the current scene framer play the NPCs in that person's scene.

    The only problem with this is what to do about scenes that involve the PC that the designated NPC wrangler is playing. The solution that springs to mind is to go around the table in order until you find the first person whose PC isn't in that scene and drop the role on them. In the event that all the PCs are in the same scene, it's probably going to be so busy that you won't need any NPCs.
