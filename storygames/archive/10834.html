\n I'm hoping some creative souls can help me flesh out a kind of oracle tool I've been kicking around.

I'm planning on running a one-shot game this weekend of modern day treasure hunters. But, with a dash of horror for good measure.

Think: Tomb Raider, Uncharted 2, Clive Cussler, "modern" Indiana Jones stuff. That kind of thing. Then, throw in some Lovecraft-style flair.

What I'd like is a situation element for each card in a deck. The idea is that each suit features a kind of situational thing, like a treasure or a bad guy or contact, etc. And, the value of each card has a progressively grander weight. So, a 3 or 4 is a relatively minor treasure, but a Jack or a King is a big deal famous treasure.

Still thinking on what, if anything, to include with Jokers.

Here's the list, which I'll edit as people contribute. Once it's complete, I'll explain how I plan to use the deck for a kind of shared situation / prep for the game:
