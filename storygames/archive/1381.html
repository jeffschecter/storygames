    I used LaTeX for laying out some documents for ERP a couple of years ago.

    Strengths are that you just write content and tag them as headings, footnotes, emphasis, etc. then fiddle with a separate template for layout. To change the layout you just change the template, no need to go back and alter your content in any way. So it can be a powerful way of indexing and laying out large pieces of text without having to actually lay out each page -- that's all done by the compiler.

    Weakenesses for me were the limited fonts (but Palatino, Times, Computer Modern, should do most people) and the fact that I like to use page layout software like InDesign.

    The level of learning required for LaTeX is surprisingly low, though. For instance, if you can write any HTML then, given an example template for LaTeX, you should be able to quickly pick up LaTeX.

    So, where would you find an example template? Glad you asked. On the ERP page there is a zip file called "LaTeX template for ERP documents. 68 kB, Zip archive (LaTeX, PDFs). 3 Nov 02" containing a LaTeX source file, a PDF of what it looks like and a logo in PDF (to show you how to embed images).

    A gamer friend from Germany picked up LaTeX real quick from these and has been doing Fudge, Forgotten Futures, etc. PDFs in LaTeX since then.

    Hope this helps.

    I've used it when I really didn't have time (or the energi) to layout my stuff. It takes a bit of time getting the needed (free) software together. I work on OS X.

    Very very easy to handle if you only use standard templates - very tricky if you want to change a particular thing, say page numbering, if you don't know what you are doing. But if you can live with a standard template with a standard font, it's a huge time saver and the finished layout is professional quality. In the old days images used to be very tricky, I think this is less so now, but I am not sure.

    The comparison to HTML, or Wiki tags, is spot on. The exception is you have to compile the file everytime you have made a change to see the result, which at first was a bit "Eh, wow" to someone like me.
