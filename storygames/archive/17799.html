    \n I've run it a few times, and definitely enjoyed it. My players are used to trad gaming fare like d&d or hero, so this was way out of their comfort zone, in a fun way. The first time we played, it took forever, because there were a lot of abysmal and mixed rolls, and people were not aggressive about recovering memories. The second time was the opposite in all respects. Super fast and short game, amazing rolls, sometimes forced to regain memories.

    \n Easy to learn, easy to teach. I've run it twice, once for a 7-year old, once for a couple of Japanese gamer friends.

    Both times were great fun. Both reached satisfying conclusions after one 4-hr session. I expect it would take more time if you have more players.

    Basically, it's good at what it does and worth the money.

    \n Ran it a few times. Good times all around. Players picked up on the dice resolution pretty quick.
    I think it best for 1-3 session runs. Most of the time my players were not concerned with memory recovery.
    Keep in mind, a game can end quickly if memory recovery is the only result. The GM needs to pace the game so this doesn't finish the game too soon.
    very flexible game, can use it for a variety of themes easily.

    \n I've only played it once, but it was a lot of fun. The stripped-down system is very easy to learn and teach quickly (I read it for the first time the night before running it), and the ticking clock of the Chasers and oracular nature of the die mechanics makes for pleasantly unpredictable emergent story beats, but it might help to already be a bit familiar with how Otherkind Dice work, which we were.
