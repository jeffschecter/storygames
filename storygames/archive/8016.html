    \n It went very well!

    We also had an action point system that fed into it. You could spend an action point to either gain a feat for an encounter that you had the prerequisites for or gain a feat for 1 round that you do not have the prerequisites for.

    Then when we would do a Burning Wheel style Trait Vote, the other players could vote to award you a bonus Feat from the list of feats that you spent action points on. If it is a feat in which you lack the prerequisites, you would gain 1 of the prerequisites instead.

    Which was a clever way to also increase low abilities (often without an ability bonus since the feat prerequisites for abilities are at odd numbers, 11, 13, 15).

    Every adventure we would start with a minimum number of action points plus additional points for completing personal quests.

    \n As far as I can tell, Keys (from TSoY) work with ANY system that doesn't have some mechanic that already does the same job, so long as:

    a) You come up with a way for people to spend those XPs, tying it in to the regular system, and

    b) Your players all "get" what the Keys are about, and you're all on the same page about the game you're playing (Creative Agenda, etc).

    I've used them, just bolted on to a simple conflict resolution system and it worked really well.
