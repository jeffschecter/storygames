So I'm trying to gear up my creative juices before the upcoming game-chef, and decided that since I'm on vacation this week I'll create a sort of anthology of 5 scifi based games which all:

  • Have strong situation that allows them to function as a pickup game.
  • Are derived from the same basic ruleset, though adapted to each uniquely.
  • Focus heavily on story-telling as well as role-play.
  • Contain a trippy classic scifi flavor.

I'm calling this little week project Five False Gods, and here is a link to my blog that holds the stubs for each game right now: Rainfall

As I fill each game in this week, It will be posted on the blog. Expect the first two to be on the blog tomorrow evening (EST).

I'm really interested in what sounds cool and what doesn't as I progress, since my tastes tend to head into the unpopular range Any feedback is appreciated.
