It would be great to have an index of Many Story Games (as well as games that are just good), and this domain could well be the place to do it. It would be a way to introduce people to the hobby: "Interested in this story games like the one we just played? Just go to the domain and click on the List o' Games."

However, extremely long lists of games are intimidating and hard to parse. Ideally, you could view this list in ways to emphasize these features:

  • breakdown by categories / description ("dungeon", "short", "live action", "noir")
  • breakdown by a single user's submissions (so you can just see Dev's short list of best pirate games)
  • rank by popularity (because you should see Dust Devils before you see Dev's new game)

What would be good, perhaps, is... something that works like Digg (users post links with color commentary, organization by tags, rank by popularity & voting) but only and ONLY for links to existing games out there.

I think this is a very good idea. FWIW, there is already Digg-like open source software existing.
