    Posted By: Justin D. JacobsonIt's not a simple answer. It was originally published by GDW, but they've licensed it out several times since then--most recently to Mongoose, whose version is due out next year I think.
    \nI think the only clarification I'd make here is, as far as I know, the Traveller IP is owned by Marc Miller and not GDW (since GDW is defunct). Marc Miller's own publishing imprint is Far Future Enterprises.

    Several other folks have a license (or had) a license to publish "versions" of Traveller:

    - Mongoose (for a new version of Traveller, which Marc intends to be "the basic game", as far as I know)
    - Steve Jackson Games (for GURPS Traveller)
    - ComStar Games (for Traveller HERO)
    - Quick Link Interactive/RPG Realms (for the D20 version of Traveller, Traveller20)
    - Far Future Enterprises (for all GDW versions of Traveller, T4, and the upcoming T5, which Marc intends to be "the advanced game", as far as I know)
