Over at Transneptune Games, we're running a game tasting this week. Different RPG each night, Monday to Saturday. We've begun by playing some Apocalypse World, which you can read about here. The whole list of what we're playing:

\n
    \n
  • Monday: Apocalypse World
    • \n
  • Tuesday: Don\u2019t Rest Your Head
    • \n
  • Wednesday: Primetime Adventures
    • \n
  • Thursday: How We Came to Live Here
    • \n
  • Friday: 3:16 Carnage Amongst the Stars
    • \n
  • Saturday: A Taste for Murder
    • \n
\n

As we play more, I'll link them from this post.

\n

In a sense, though, this isn't actual play. We're playing them for fodder for thinking about game design, so we are going to write more about that than about how the game went as such.
