\n In this thread on a spanish language game Graham mentions a frustration I often share in not being able to read an interesting sounding game because of language barriers

I'll sometimes pre-order games but I've never participated in a ransom model funding for a game. But I started thinking that I would be willing to put in $$ to allow more story games to be translated (presumably into english for selfish reasons rather than from english)

Some of my favorite game designs have been translated- like until we sink, wuthering heights roleplay, de profundis, so I'm wondering what else I'm missing

so my question for folks who know the non-english language game scene- are a decent percentage of interesting games being translated?

is there a need to think about sources of money for more translation? (I'm assuming small pots of money to help gamers do the translations rather than outsourcing to full time translators)

thanks for any thoughts

    \n So there's this thing, right, where Chinese roleplayers translate game material by committee, on forums, and then release it as free PDFs for everyone to use. This is, aside from the core D&D books and a few other things, the only way to consume RPG material in China. I would love, at some point, to see if I can convince some indie game designers to donate PDF copies of their games to the Chinese roleplaying community, to be translated and freely distributed in the Chinese language (because it loses you no sales). I started translating Dogs for them, but then quickly realized it would be much easier to just give them copies of stuff for them to translate on their own. But, first, I'd need to get authors' permission.
