\n Kat and I have been in touch with GenCon about securing table space and promotional opportunities for the Indie Games Explosion and Games On Demand. I\xe2\u20ac\u2122d like to discuss these two things separately, as the registration deadline for events is very early in April, and we\xe2\u20ac\u2122d like to get these finalized as soon as possible.
\n
\nINDIE GAMES EXPLOSION
\nThe Indie Games Explosion at GenCon is a collection of scheduled events of independent RPGs being run at the con. If you would like to run a scheduled event as part of the Indie Games Explosion, please contact Kat\xe2\u20ac\u2122s e-mail (whim at enter dot net) as soon as possible. If you run enough games, you can have the cost of your GenCon attendee badge refunded (does NOT apply to exhibitor badges). At the present time, only one person has sent us events to be included in the Explosion. This looks like a tiny Explosion compared to previous years, and we invite you to join us and run some great games!
\n
\nGAMES ON DEMAND
\nGames On Demand was a big experiment last year. The idea is simple: Show up at a table with an indie game you want to play. Maybe there will be some folks there willing to play it with you. Turn in a generic ticket and PLAY. At GenCon 2006, we learned about how the idea worked in practice and hope to expand it this year.
\n
\nGames On Demand works because of the Table Custodians who collect the tickets, run some of the games, connect players with GMs, play in some of the games, and generally have a good time. However, at the moment, NO ONE has volunteered to be a Table Custodian for Games On Demand 2007. This is a big problem. Without Table Custodians, Games On Demand cannot function.
\n
\nThe Games On Demand tables run on three 4-hour shifts on Thursday, Friday and Saturday\xe2\u20ac\u201dthey are 10am to 2pm, 2pm to 6pm, 6pm to 10pm. Sunday has a single shift from 10am to 2pm. That\xe2\u20ac\u2122s ten shifts. If Kat and I each take one, that leaves eight unfilled shifts. If we don\xe2\u20ac\u2122t have volunteers for those shifts by next Sunday, March 25, there will be no Games On Demand at GenCon this year.
\n
\nFurthermore, if we are only able to get one Table Custodian per shift, Games On Demand will operate on the same scale it did last year. If we want to expand, we need more Table Custodians.
\n
\nIf you want to be part of one of the coolest events at GenCon, you can sign up for a specific shift in this thread, or by e-mail to Kat (whim at enter dot net). Thanks!
