Dreamation 2008
January 24-27, 2008
Hilton Woodbridge, Iselin, NJ, USA

Thursday 8:00PM

Universalis. Run by Nicholas Marshall
Contenders. Run by Jeff Collyer
Shock: run by Joshua A.C. Newman
Misspent Youth. Run by Robert Bohl

Escape or Die! Run by Fred Hicks
Kingdom of Nothing. Run by Jeff Himmelman

Grey Ranks: German Tanks vs. Home-Made Rifles, Jason Morningstar
Serial "Bind Them, Torture Them, Kill Them." Run by Kat Miller
1001 Nights. Run by Meguey Baker
Questers of the Middle Realms: Ko-Sha on Ice. Run by Clinton R. Nixon

PTA : Super Hero Edition. Run by Jeff Collyer

Ganakagok (revised): The Edge of the Ice. Run by Bill White.

Beowulf: playtest run by Joshua A.C. Newman
Young Adult RPG: Playtest run by Remi Treuer
Jeepform, "The Upgrade!" Run by Tobias Wrigstad and Emily Care Boss
Polaris. Run by Matt Weber
With Great Power.... "Crossover" Run by Kat Miller

Friday Midnight

Monster and other Childish Things, "What You Got for Christmas." Run by Jeff Collyer
Coming of Age "Space Police Cadets" run by Mendel Schmiedekamp
Escape or Die! Run by Fred Hicks

Don't Rest Your Head. Run by Ryan Macklin

Artesia: Adventures in the Known World, "Doom of the Fallen." Run by Don Corcoran
Burning Wheel "Quest for the Holy Grail" ; run by Mel White

Fatal Fantasy (playtest; FATE variant): The Fire of Analla. Run by Bill White.

Coming of Age "Space Police Cadets" run by Mendel Schmiedekamp

Spirit of the Century: Rex Rich and the Warlord of Mars. Run by Bill White.

Escape or Die! Run by Fred Hicks

Under my Skin (free form playtest). Run by Emily Care Boss
1001 Nights. Run by Meguey Baker

Burning Wheel: A Damsel in distress. Run by Nicholas Marshall
Project Bearclaw: Tastier than a Donut (Jason Morningstar and Kevin)
Serial "Bind Them, Torture Them, Kill Them." Run by Michael S. Miller

A Penny For My Thoughts. Run by Ryan Macklin

Sunday 1:00PM

Indie Game Designers' Roundtable. Moderated by Fred Hicks and Jared Sorensen (and possibly Luke Crane).
