\n Maid RPG Random Random Event Table Table

Roll 1d6:

1: Roll on standard event table (page 26) (For non-standard settings, roll on world-specific Events)
2: Roll on world-specific event table (page 115-122)
3: Strange Weather! Roll on Weather Table (123)
4: Roll on appropriate Theme Event Table (page 112-114)
5: Roll on a random Theme Event Table:
\t51: Romance Table (112)
\t52: Light-Hearted (112)
\t53: Action (113)
\t54: Hard Life (113)
\t55: Dark (114)
\t56: Horror (115)
6: GM's Choice! Roll on any table you want!

    \n Great minds think alike, I guess. I'm working on a "Bizarro" World with an accompanying Random Event table. The first half is references to every other Event table (including the Supers, Gothic, Melodrama, and Intrigue ones I'm working on) and the second half is the most surreal stuff I could think of ("A random maid vomits up a random Item"). I also want to take a cue from the Toon Ace Catalog and do a "Table Table."
