\n First post! Hi everyone.

I wanted to play a game like Dungeon World in a Traveller-esque sci-fi setting. I saw Uncharted Worlds and read it, but it wasn't exactly the game I wanted to play. It's great, but I wanted something just a little more crunchy, and a little more DW.

So I started designing, and I think the latest version of Offworld is ready to share. A lot of it is very essentially Dungeon World, with some Freebooters thrown in for good measure. It does a few things pretty differently, too.

Many parts still need fleshing out, and I need to test the heck out of it, but here's what I have so far. Thanks for checking it out!
