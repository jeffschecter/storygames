\n Ice Station Nerdly is a mini-con I host in my house in Alexandria, VA. I host it each Fall, and this year it will take place on November 6, 2010.

To make it easier to set up the Con this year, I am using Conplanner.com. It was created by the Fear the Boot guys to run their annual Fear the Con. It allows participants to create and join events at the con, so I think it will make it a lot easier for people to set up games. It also has RSS feeds, so you can keep abreast of any changes to activities or the Con.

You will have to create an account on ConPlanner. As before, Ice Station Nerdly will have two slots, afternoon and evening, but we have 5 potential locations for the games. (We added a screened deck to the house. Weather permitting, of course).

After you create an account on the site, you can join or create activities (I have created one). I will assign locations to the games (living room, dining room, basement, upstairs, deck).

Hopefully this will make it easier for us to set up the con. If you have any questions or problems, please contact me at jhosmer1@gmail.com

The Conplanner site for Ice Station Nerdly 2010 is: http://www.conplanner.com/ISN2010

We also have a Google Group, so if you are interested in learning more about Ice Station Nerdly, or to make sure that you get announcements about future Ice Station Nerdly events, please whisper me your e-mail address and I'll send you an invite.
