\n Posted this idea on my LJ a few days ago. I thought it might be of interest to the writerly sorts on SG. \n
Story Squared*

Write a number of short stories, all of which have a word-count equal to the number of stories in the series.

For example, if you were doing a 100x100 challenge, you'd write a hundred short stories, each of which is exactly 100 words.

If you were doing a 25x25 challenge, you'd write 25 stories, each of which is only 25 words.

If you were doing a 6x6 challenge, you'd write six, six-word stories.
\nThis is sort of a "50 Pounds of Clay" exercise to work out some creative blocks and hopefully create some cool flash fiction in the process. This would be handy for for projects that call for lots of little bits of flavor text.

As I mentioned in the original LJ post, what intrigues me is the game-like tension of deciding what challenge you'll pursue. You can choose to write fewer stories, but that severely restricts your word-count. You can choose to increase your word-count, but that quickly means you'll have to write a LOT more stories. You can also "squish" the challenge to make it taller (write twice the number of stories than the word count in each story) or wider (write number of stories equal to double the word count in each story).

Anyhoo, thought some folks might find it an interesting challenge.

* Credit goes to Ryan Macklin for coming up with the name "Story Squared." I'd given the idea the less approachable name "NxN Challenge.") ^_^
