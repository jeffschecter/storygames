I've finished my first major revision of my AGE Model for gaming, and I want to do some exploratory work in seeing if the model relates to - you know - reality. The first step that I want to take is to record a roleplaying session (or find a recorded one), transcribe it and code it using the model. This requires (first of all) that I have a session to transcribe. Aside from the Dead Alewives D&D sketches, does anyone know of honest-to-goodness actual play that's been recorded?
