\n I'm making a game about cats, and would love some eyes on it.

I've gone into this hoping that the character sheets would kind of tell me how to write the rest of the game. I want the sheets to convey a self-contained set of rules for each character, to communicate how to play the game as much as possible. I think there will need to be an additional sheet with procedure for setup and other order of play things.

The dice-as-resources concept is very untested, and I need to run some games to see how the dice pools should be adjusted, or if explicit scene framing will even work here.

I have pretty specific ideas about the tone of this game, but serious questions about how relatable that tone is / whether it fits with the kind of "cat stories" other people are interested in.

So, here's the first seven (out of ten, I think) character sheets:

http://tenthousandmogs.com/wp-content/uploads/2014/12/Cats-charactersv0.1.pdf

    \n Thanks for looking, all.

    @BeePeeGee, the rules are an adaptation of Otherkind dice, but deployed asymmetrically so each character has a slightly different set of mechanics for interacting with the fiction.

    As to intention, I have the same concerns. I hope that the character "Introduce Yourself" sections fuel interesting interactions, but the basic situation will need a strong in-play primer.

    Hence, the first draft of the "Prologue" rules: http://tenthousandmogs.com/wp-content/uploads/2014/12/Cats-prologuev1.pdf

    Hoping to give enough of a prompt to set tone, establish a few basic premises, and spur imagination without pre-defining too much. Does it do this? Is there anything you're left wondering about that would obstruct play?

    \n I've been thinking about a game like this for a long time - although my take on it was more realistic (and therefore kinda ironic, like "You see a huge monster coming - you do an action to escape it - but actually it's just a car.") But I like where you're going with this - a much more fecund fantasy setting.

    The "Warrior Cats" series definitely comes to mind, I assume you've read those. A couple other influences you might want to think about if you haven't already...

    "Jenny and the Cat Club" - classic YA books by Esther Averill

    The secret life of cats in some of the H.P.Lovecraft "Dream Cycle" stories (in Lovecraft's world, cats not only have a society but they can actually leap to the moon from rooftops, and they have a whole civilization up there)

    And a lot more here: http://en.wikipedia.org/wiki/List_of_fictional_cats_in_literature

    Keep going! I would definitely play this game. :-)
