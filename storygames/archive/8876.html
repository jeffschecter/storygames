    \n Ash doesn't self-promote much, at least not here.

    I played a demo last year. It's pretty good. Great setting and a competent system. It uses playing cards. I'm moderately enthusiastic about it - note my slightly restrained tone, because the system is workable rather than massively exciting - but, yes, it's worth having a look at, especially if you like that rural horror theme. Which I do.

    One interesting thing is that "rural horror" seems to encompass both rednecks in the American South with shotguns and hicks in the British West Country with pitchforks and, bizarrely, the combination works.

    \n First off, thanks to Graham for tipping me off about this thread. I've never been very good at promotional stuff. In case anyone hadn't guessed, I'm Ash, the author of Fearmonger. If you've got any questions please feel free to ask.

    It's funny this thread should come up now actually, as I just started working on a kind of sequel / spiritual successor to Fearmonger. It's still in the preparatory stages, but since it's using the same basic system with a few tweaks, it shouldn't take so long to develop. It's very much its own game though, and the tweaks in the system should emphasise quite a different style of play. I'm quite excited about it so far.

    Not that I've gone off Fearmonger though, I'm still very fond of that game. That's one of the reasons I want to expand it in a new direction, and appeal to fans of more modern style horror.
