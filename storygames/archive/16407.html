\n WotC is doing a big Drow push this summer. And, I'd like to relive some of the glory days. My plan is to run Vault of the Drow as a stand alone, so it will require a little massaging. My current, preferred systems are Dungeon World, FATE, and Cortex Plus. Do you think I could pull it off with one of those systems, or I would I be better off sticking to the original system, or a retro-clone? I don't mind drifting, but I don't want to end up with an 80/20 prep to play ratio.

    \n I've run a ton of AD&D adventures with Dungeon World. They're my go-to for convention games. Really, you just need to divide the adventure into four bits;

    - maps
    - fronts
    - monsters
    - treasure

    Take the maps apart and erase any rooms you don't like or add in some new ones but leave them blank for character input and exploration. Create relevant fronts for the big stuff and make sure you have DW versions of all the monsters you'll need. Clear out all the "+1 swords" and replace them with cool, custom magic items. Write a few customs moves for the adventure, throw together a little starting setup based on the characters (give them all a reason to be there) and away you go!
