      \n We did it! Some are still doing it. It was fun!

      I met Colin, Owen, John, and Carl, along with a few others whose names I did not catch. Friends! I was the kid with the square glasses and the beanie.

      I facilitated a game of Hell for Leather, which went swimmingly (I will detail that in its own thread so Sebastian sees it).

      John Kim ran a game of Mouse Guard which was totally rad, except I made a mistake in character recruitment and made myself a bit too independent for a one-shot game about teamwork. :P I still had a great time--looking forward to being at John's table again!

      Anyway, I'm sure there will be more stories soon. I'll pop back in then and talk more.

      Thanks, story-gamers!

      EDIT: And Game Kastle in Santa Clara totally had the Adventure Burner, which I broke down and bought. Not that it was much of a fight; I have the Instinct "Always buy Burning HQ products."
