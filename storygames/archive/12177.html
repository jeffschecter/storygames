    \n Basically, you've got the theme laid out for you. There are some dials that each play group will customize. The group sets the tone, and a few samples are provided along with how that works in play, and you also set the magic level. The implications of each magic level are laid out. Then there is a set of villains and other supporting characters all statted out for you.

    The players make characters, hook their passions into the setting, and you can start right away. I'd say total prep time would be around 30 minutes. Sometimes it can take a while to build a good theme document. I've had the first session of a Mortal Coil game just be theme and character preparation. This is a sort of plug-and-play setting.

    This is also the first of a series. Another one will be released next month, and if they do well, I plan on putting them all together in a book and publishing them, once I have enough. Each theme will be from a different author.

    Posted By: Paul T.Is this something that makes sense to play once, like a module (even a campaign-length module), or something that gives you material for several different games?

    I'm curious about the design decisions that went into this!
    \nThe tone and magic dials make it easy to play a different game each time. Depending on where you set the dial, the setting changes a great deal. I think you could get a lot of re-use out of the supplement.

    I wanted something that would be useful for Mortal Coil players. Since it's such a toolkit game, creating any additional material for the game was a bit of a challenge. I do think starter settings are a useful idea. We'll see if any game fans agree!
