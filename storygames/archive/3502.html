\n There's been some discussion on various threads (too many to thoroughly track down) about the value of taking short, one- or two-word events, and pulling them together to create plot.

If you're pursuing something like this in your designs, it might help to know that the Internet Movie Database has started tagging movies with plot keywords. They're not always very accurate, but there sure are a LOT of them. I can't seem to find a single universal list of all the keywords available, but if your game has some movie inspiration, it might be valuable to look up the keywords of that film.
