    \n I used MapTools to run a PBeM game for about 6 months. It stopped about 6 months ago. I'm quite interested in VTTs and so far MapTools is far and away the best. It's free, cross-platform, system independent, fairly easy to use at its base but quite powerful. It also has active support, both by the creators and fans. New builds come out quite regularly and people are making all kinds of material for the game system of your choice.

    The only tricky technical part (and this is the case for any VTT that is hosted by an individual) is setting up your Port Forwarding properly. If you have a bit of understanding of how your router works at home, this shouldn't be a problem and there is tons of help at the MapTools forums.

    I don't know how up to date there are, but there are several really great tutorial videos on MapTools that go through the basics and give a good overview of all the various features.

    Let me know if you have any specific questions.

    \n I'm not sure if this has changed, but if you want a Mac version that integrates into Cocoa (or whatever the hell the Mac GUI is called, I get them all mixed up), you used to have to wait a bit. So you can run the latest version on the Mac, but for people used to that interface, it's much better to get the one that starts up properly and has the Mac menus and all that. It isn't a big deal, just a question of making sure you get the right version between all the players and being one or two builds behind is not a dealbreaker.

    Maybe they've fixed it and integrated the Mac version into the current release, as the time differences were getting shorter and shorter when I was using it regularily.

    Again, I am reminded of what a dynamic and active community that is. They seem to have good, productive and positive leadership buoyed up by active, productive and positive fans. Check out this thread on speculation about future UI changes to the program as an example of how positive and dynamic the web can be when the right people are working on the right project with the right attitude. Quite inspiring!
