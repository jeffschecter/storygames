    \n One relatively small thing that I think needs to happen - if there are any panels discussing indie RPGs, we definitely need to make sure that those games are for sale at whatever booth is there, selling games.

    There've been a ton of times, every PAX, where people hear about a game, get fired up and want to buy it and then ... no dice, 'cause it's not RIGHT THERE RIGHT NOW.

    \n\nApril 2011 \n
    \n
    \n
    \n
    \n
    \n I attended PAX Prime last year, all the way from Australia, and spent a lot of time in the gaming area - highlights included my first game of Fiasco with Sage, and a game of Lady Blackbird which went off the character rails a smidge, to say the least. It was probably the single most concentrated dose of gaming I've had in my life (haven't been able to start any real games down here, or find a decent community for it), and it was probably the best means by which I learned a whole lot about both playing games, and running games.
    \n
    \nI'll be heading up again this year, which will be swell, but I think my primary idea is to maybe set up scheduling a bit better, or pre-planning times for games. Even just a more formalised version of the loose "hey, we'll be running [game] at this time, put your name down if you want to play" system that was going on last year. As much fun as it was hanging around with everyone in the games room, it'd be cool to be able to have a time in my head as to when I'll be getting to play the latest indie sensation.
