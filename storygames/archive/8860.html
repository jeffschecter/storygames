So, I'm working on a Storming thing, and I wanted to expand the monster powers a little. How do these guys look?

\n

Delay
\nMonsters with this ability have a limited capacity to plan ahead. The monster gets two green dice to prepare an action it will enact next round. Name that action when you use Delay; the hits for your Delay become bonus green dice for the next round's action.

\n

Metamorphosis
\nMonsters with this ability can take more than one shape. Build each shape separately: make two monsters with the same materials, giving each one the Metamorphosis ability. Instead of choosing freely, give one monster one material's exact Endurance value, and give the other monster the other value. Finally, for every two abilities each monster has besides Metamorphosis, one must be the same. That means if you're making a minimal monster with Metamorphosis, then it has these abilities: Metamorphosis, Ability A, Ability B / Metamorphosis, Ability A, Ability C. If you add two abilities: Metamorphosis, A, B, C, D / Metamorphosis, A, B, E, F. If you add an odd number of abilities, the odd one can be different for each.

\n

It's the monster's special move for the round to change forms. Track each form's damage separately.
