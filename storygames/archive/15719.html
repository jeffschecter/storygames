\n I highly recommend this app:
http://www.fnordistan.com/smallroller.html

It calculates probabilities on any number of dice, dice types, dice pools and includes options for modifiers to each die or the total, as well as options for determining odds for rolling equal, lower, or higher than the target result. Good for most RPGs.





The stand-alone executable file is still Windows-only. However, the programmer has included an image file that will run on multiple Operating Systems as long as Cincom's VisualWorks is installed.

This app is 7 years old but I still frequently use it.
