    Sure!

    First download an IRC client of some sort (If you need help with that, ask, and provide your OS, and I'm sure we can hook you up).

    Then you need a server to connect to. Each client connects in slightly different, but generally intuitive ways.

    Once you've connected to a server, you need to join a channel with people in it to talk to. Channels are prefixed with a pound sign ('#'). For instance, I spend time hanging out in a channel called '#indierpgs'.

    You need both a server and a channel because each server is self-contained (meaning a single channel name can exist independently across all IRC servers), and because channels help you keep discussion groups at a manageable size.

    EDITED: Wait, are you asking for a way to use an internet browser to connect to an IRC server via some sort of applet-like thing?

    Thomas
