\n I just picked up this ancient (ca. 1970) Canadian "family" board game off ebay. It's weird the extent to which it anticipates a lot of the gameplay features of later, more specialized storytelling games of the types more often discussed in this forum. The fundamental mechanic is a lot like ONCE UPON A TIME, but there are also some quite witty, well-written "prompts" to get new stories started. Anyone else ever come across this old chestnut before?
