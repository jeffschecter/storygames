\n THE SETUP:

Okay so i was sitting in the bus going to work, trying to keep my mind off the pain of the massive sunburn I got this weekend. I was thinking about the games i really enjoyed Gm-ing. Danger Patrol came to mind. I'm a big fan of the reaction oriented mechanics of the game and how easy they allow me to run it on-the-fly. and then... EUREKA! I realized how cool DP would be to run some Silver-agish four-colors Superhero game.

THE QUESTIONS:

1)Do you think it would work?
