\n Hey all, I've been working on a card game where the cards have lists of generic story & rpg elements that mix up in play for all kinds of goodness.
By generic I mean they work in just about any setting I can imagine - for example, there's no "astronaut", but there is "pilot".
There are roles (soldier, pilot...), places (mansion, graveyard...), items (cash, weapon...), events (holiday, fire...), adjectives (strong, wise...), and relationships (parent, spouse...).
Now, although I've already filled up the deck with my ideas, I want to make sure that I have the very best story tropes/cliches in there ... so I thought I'd throw it open to you guys.
But should I throw out my list first, so we don't repeat ourselves, or just let you guys go at it, unpoisoned by what I've already got? On the one hand I wouldn't want to read someone's big list, on the other hand I wouldn't want to contribute if I felt like I might be redundant...
