\n I was casting around for a system to use for an short run SciFi game I'm going to start shortly and I remembered want to try out TSOY. A couple of questions. The setting will be pretty firm sci-fi (in the style of Iain M Banks): Has anyone done this? Do any experienced TSOY users foresee any potential conflicts between system and setting tone with this setup? I'm using the online rules; they don't seem to have the costs for spending advances. Is this intentional or am I just failing my spot check?
