    So we played Drowning and Falling and had a good time, although I'm fairly certain we played wrong!

    We couldn't tell from the text whether every character in the party was supposed to face the same challenge in every room, or if only the character "in the lead" was supposed to deal with it. After sending one character through one challenge, we estimated based on time that only the "lead" character faced each challenge. With 30 cards' worth of challenges and six players, with only one character facing each challenge, our game filled the two hours of time we had for it.

    But there wasn't at all as much character death or character interaction as we expected there would be, so I think we were supposed to have every character face every room challenge. Am I right on that?
