\n Hi

I bought a Sony Touch Edition recently, for a number of reasons, one of which was that I could view some of my RPG pdfs on it. The result have been mixed. Some pdfs can be read reasonably well and some are reader unfriendly. The reader has a few more viewing options than the Kindle which is why I bought it. I was wondering if any publishers or designers were releasing products in epub format.

I am aware of software like Calibre that allows you to convert. I had a go with a 100 page pdf I had that was beautifully designed. Basically I would have had to spend a number of hours reformatting the resultant mess into something usable. I'd rather buy something completed than spend my time reformatting at a computer screen. I bought the Reader specifically because it has a no-glare screen FFS.

    \n The thing about epub is that you have to mentally let go of certain things. You don't control the resolution. You don't control the pagination. You probably don't control the fonts. If you are used to micrometer control over layout, it can be difficult to give such things up. But epub isn't that kind of party. It is not at all about presenting the user with a work of art. It's about presenting the user with data that they control.

    Even given that, the two things you mention are not particularly difficult to control just by changing CSS and not altering the HTML at all. While I could see that "an extra line break between the end of the recipe and the name of the next dish" is difficult, doing something in CSS that has the same visual effect -- like changing the "recipe" style to have a larger bottom margin -- is not and is more correct besides. Since InDesign will build you a file using named styles from its own style sheets, it should render the HTML just fine, and in a state fairly easy to tweak with CSS overrides. (Granted, the rendering in InDesign could do a couple of small things to make this much easier.)

    The other thing InDesign does is allow you to base the ebub off the outline, rather than using the flow of the text. Since you can pretty much adjust the outline without effecting the layout within InDesign much, this is usually a lot easier than making the text flow work right, particularly for stuff like placing sidebars, images and tables within the flow.
