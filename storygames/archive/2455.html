dyjoots

For those who don't know, So Long Sucker is a game invented by John Nash (the guy from A Beautiful Mind) and a couple of friends. The rules for the game can be found here.

Now, I've been working on and off on an RPG (found here) that involves some very similar aspects: extremely simple rules, competition between players, deal-making and deal-breaking. Of course, my game isn't quite as elegant (or brutal) as So Long Sucker, but I'm definitely looking at involving the basic concepts... I'm not entirely sure how it would work yet, but the draw of such a game is incredibly strong.

Has anyone ever played So Long Sucker? Any thoughts on mixing role-playing concepts with it or other political/intrigue games like Diplomacy or Intrigue?
