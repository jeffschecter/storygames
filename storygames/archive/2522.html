    Page 24: Agreement errors in several Latinate names. I don't think it matters even slightly for this particular list.

    Page 25: No names display.

    Page 54: Still has dummy copy.

    Page 64, Male, entry 12: There's a dot after it.

    Page 74, intro: Don't capitalize the E in "e.g." here, you're not starting a sentence. Use an em-dash after "Bobbies," not a hyphen.

    Page 79: The headings could probably use spaces.

    Page 80, Surnames: List numbers are doubled.

    Page 87, intro: see notes for p. 74.

    Page 94: Hey! Only nineteen entries!

    Page 97: Intro text should probably not be followed by that horizontal rule.

    Page 101: Punctuation and capitalization under "Surnames" may need some stylistic tidying.

    Page 110: List is numbered 21-41.

    Page 115: Under "Food Notes," use em-dashes.

    Page 125, intro: Convert double-hyphen to em-dash.

    Page 128, intro: Em-dashes.

    Page 130, Surnames with "de": Double-check commas and quotes.

    Page 131, Abbreviations: See notes for p. 130.

    Page 139, intro: Need space after "information,"

    Pages 144-272: Doubled numbering on all lists. A few entries here and there are not double-numbered.

    Page 155, intro: Convert hyphens on either side of "but not instead of" to em-dashes.

    Page 158, intro, 3rd paragraph: If we're talking about the conjunction "y," maybe it should be in quotes.

    Page 162, intro: Check commas and quotes.
