What's the Deal Behind This
This is not a contest, this an open event held by the community for the community. All the files can be found in this google docs folder:
https://drive.google.com/folderview?id=0B0bUHr-C_JJ0VTdvaUQtLUNWZHc&usp=drive_web
Any promotional material that anyone creates can be put into this folder for everyone to use. The links and the information submitted for each project will also be publicly available.
RPGenesis has no centralized source that everyone has to link to. Any blog, forum, site, podcast, channel, etc. can take all this stuff and help promote the event. If possible, just try to use the #RPGenesis2014 hashtag for cross-reference.

Writing Games Since 2010
The first RPGenesis was organized by just a couple of people in Portugal. We wanted to do a week long NaNoWriMo for RPGs and so we did. On the next year, the Brazilian community joined with the Portuguese and the event grew. After that, we added the Editor's Week as a way to open the event to all kinds of talent. Also, we would like to welcome non-Portuguese-speaking game designers to see how far RPGenesis can spread its wings. Let's go global!
