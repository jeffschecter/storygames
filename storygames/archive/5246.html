    \n Locate your Firefox user profile folder (see this example)

    In the 'chrome' folder create a file called userContent.css

    In the file add the following\n

    @-moz-document domain(www.story-games.com) {

        CommentBody.a {
            color    : blue !important;
            text-decoration : underline !important;
        }

    }
    \nThen restart. Links in forum posts should be blue and underlined
