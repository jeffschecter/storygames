    \n Yeah, it's tricky, and I have no idea why they moved to such a dumb search functionality. I need to write a FAQ at some point. I actually had a thread way back when about how the new search works... but you have to search to find it now.

    Okay, so here's kinda how it works:

    If you want to absolutely return results of a particular word, you put a plus in front of it:

    +Andy

    If you want a particular string, like "in a wicked age", you put it in quotes and add a plus in front:
    +"in a wicked age"

    If you want two strings, like "I remembered Andy said something once about In A Wicked Age..." you go:
    +Andy +"in a wicked age"

    ...but it's kinda fucked, so let me see if I can get Vanoogle (replaces the SQL raw search engine with Google) on this thing working. That's probably the best solution. Because to date the best solution has been to use google with "site:story-games.com"
