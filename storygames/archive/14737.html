\n Wanted: A web tool that does this....

1. A defined group of people can all see the same document (or a scan/image of a document, but an actual editable document would be best) in real-time.
2. A user-defined set of tokens can be added to the document, removed, and dragged around on their own layer. Preferably these would be added/managed from an on-screen palette/"drawer". There is concurrency management so that only one person can change the document at one time. It would be nice, but hardly necessary, if some of these tokens were "dice" with run-once random values.
3. There can be multiple tabs of this document+tokens environment, and some tabs can contain other things - like an image, or a Google Doc.
4. Requires minimal scripting or tech savvy to set up and operate.

This seems like a "virtual tabletop" kind of solution, but all the ones I have seen are absurdly more complicated than this and make a bunch of assumptions about how they will be used that I don't care about. I have heard of people using VASSAL to do something like this, but I am not interested in the massive learning curve involved in getting to that point.

Basically, what I want is a specialized sort of markup/whiteboard tool with a few special characteristics to use as a reminder of game-state.

Any ideas? Alternatively, educate me about why I want to bother with a VTT?

    \n The other half of the solution is Graham's Catch Your Hare dice roller.

    http://www.catchyourhare.com/diceroller/

    It allows you to roll a variety of colored dice that you can drag around the play area. You can also add basic labels and share the room/view with other interactively.

    Now all you need to do is to sweet talk him into adding tokens and a background image loader. :-)

    Open RPG has some of those capabilities, but no one screen dice the last time I looked (although it does have a dice macro in the chat box so you could add dice tokens as determined by the text dice rolls.)
