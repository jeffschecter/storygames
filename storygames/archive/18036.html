\n I've launched a Kickstarter to develop a RPG toolkit app for tablets. It stores & manages your game data or character data for use at the table on your mobile device and synchs via Dropbox/Ubuntu One
It's game system neutral but will have templates and support for Dungeon World, Pathfinder(SRD), Fate (OGL) and I'll do a series of screencasts tutorials for making and sharing your own templates.
The desktop (Win, Mac, Linux) app is in beta and wil be released to backers at the end of the kickstarter for a last round of playtests before porting to Android/iOS tablets.

    \n We start with a conversation about what features we can and canot include in a skin with the game owner (or OGL/CC doc). The scope being 'a pretty template' to a full character generator.
    Design and layout happens in Photoshop. Features are coded in Unity/Adobe Air. Packaged up to be downloaded like DLCs. We'll get feedback from the game owner before we release it.
    My intent is for a skin to be a mini-app focused on a specific RPG.

    \n Templates are easy. You create people, places, etc as normal then save the file using 'export to template'.
    No programming required.

    For example:
    I'm adding a weapon template. I'll create a new object, select the type 'thing'. I name it 'Weapon'. I then add all the stats names and values (if I want default ones) that I want. Then I export to template.

    When I load this file in and create a new 'Thing'. Weapon will be an option I can choose to start from. I'll give it a new name, 'Long Bow' and change the values if needed.
