\n Since the contest is over for which FUBAR is a submission, I'm doing the rounds and letting people know that I've completed it.

The tests I've run so far have been reasonably successful. Play examples have been added in. The text has been edited for grammar (but note that this is Australian grammar, and I'm also sure there's going to be extra aspects that haven't been picked up).

Now that I've worked out a bit more about my PDF conversion software, the current release is a pdf compressed to "onscreen" resolution, so its size is drastically reduced from over 50Mb to about 5.5Mb.

Nothing's ever 100% perfect, but it's time to move on to new projects (unless demand calls me back to this one).

To those people who helped out, thanks.
