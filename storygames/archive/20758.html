\n I specially like Night's Black Agents but, in the campaign I've played, we consistently bumped against what we feel is the excessive granularity of the skill variety, namely in the investigative abilities. It almost seems like the game is aimed at a "I want to use this ability for the next scene" style of play instead of being fiction-first.

So I've been thinking about just merging these abilities into half their number and just dividing the build points by two. Has anyone tried this already? Am I forgetting something that may blow up in my face?

Cheers!

    \n We had this issue with the 2nd edition Esoterrorist rules where some of the skill descriptions struck us as odd in terms of what was explicitly not covered in one area. It wasn't so much "I have forensics, so I want to use it" -- that's something we're fine with. It was more like "I have this skill that you'd think would include X, but apparently not, as it says it's only under Y."

    If you're folding things, be aware of what is going to change. I don't mean nail down exactly what each skill does; I think you want a certain amount of overlap and vagueness. But, you want to look at:

    * Spotlight time -- the idea is that everyone should get some time to shine. Whether this means every session, every hour, every scenario, you don't want some players grabbing all the action while others wonder if they're even in the game.

    * You're halving the number of points? How often will players need to spend for best results? That's what the points do. My concern is that if you cut 20 skills to 10, say, and I now have 11 points instead of 22, say, the number of times I may want to make a spend hasn't necessarily gone down. It's just that the number of things I can use to make a spend has been halved, which isn't really a bit deal, and I now have fewer resources, which is. I'm wondering if you might want a reduction to 2/3 instead of a half.
